# 框架候选清单 — 《庄子》

以下为从《庄子》原文中提取的思维模型/决策框架/推理方法候选，共计10条。

---

- id: f01
  title: 无待逍遥法
  type: framework
  source_chapter: 内篇·逍遥游
  source_quote: |
    "若夫乘天地之正，而御六气之辩，以游无穷者，彼且恶乎待哉！故曰：至人无己，神人无功，圣人无名。"
  summary: |
    这是庄子提出的人生最高境界框架。其核心逻辑是：世间一切"有所待"——大鹏待风，列子待风，凡人待功名、待他人评价——都不算真正的自由。真正的自由是"无待"，即不依赖任何条件。
    具体操作路径分为三层：第一层"无己"——忘记自我与社会角色的捆绑；第二层"无功"——不追求世俗功业；第三层"无名"——不在乎外在名声评价。
    当一个人彻底放下这三重依赖，就能"乘天地之正，御六气之辩"，游于无穷。这是庄子哲学的终极目标，为人生重大选择提供了"是否有所待"的检验标准。
  tags: [自由, 无为, 逍遥, 决策框架, 人生哲学]

- id: f02
  title: 道枢法（两行枢中）
  type: framework
  source_chapter: 内篇·齐物论
  source_quote: |
    "彼是莫得其偶，谓之道枢。枢始得其环中，以应无穷。是亦一无穷，非亦一无穷也。故曰：莫若以明。"
  summary: |
    "道枢"是庄子提出的认识论枢纽。当面对是非、对错、善恶的二元对立时，常规思维试图判定谁对谁错，但庄子指出：是非双方如同一个环的两面，执着于判定任何一方都陷入无穷无尽的争论。
    "道枢"的运用步骤是：首先识别"彼"与"此"（或"是"与"非"）这一对立结构；然后超脱这个对立结构，找到两者的共同根基（道）；以此根基为枢纽，像站在圆环中心一样，应对环上周而复始的无穷变化。
    简言之：不与对方争辩谁对谁错，而是跳出一个更高的视角，看到这场争论本身没有终极边界，因此不值得执着。
  tags: [齐物, 是非超越, 道枢, 认识论, 决策框架]

- id: f03
  title: 以明观物法
  type: framework
  source_chapter: 内篇·齐物论
  source_quote: |
    "道隐于小成，言隐于荣华，故有儒墨之是非，以是其所非而非其所是。欲是其所非而非其所是，则莫若以明。"
  summary: |
    "以明"是庄子提出的认知清空方法。其核心逻辑是：大道被"小成"（局部、片面的成就）所遮蔽，至言被"荣华"（浮华辞藻）所遮蔽，所以儒墨等家才产生是非之争——彼此都以对方否定的为是，以对方肯定的为非。
    打破这种循环争论的方法是"以明"：即用空明澄澈的心境，不是去分析、评判、辩论，而是直接观照事物的本然状态。如同水面平静时能照见万物，人心空明时才能看到真实。
    操作要点是：在发表判断之前，先暂停自己的立场预设和情绪反应，以"无知"的心态去接近事物本身，让判断自然浮现而非逻辑推演。
  tags: [认知清空, 以明, 齐物, 认识论, 心斋同类]

- id: f04
  title: 心斋法
  type: framework
  source_chapter: 内篇·人间世
  source_quote: |
    "若一志，无听之以耳，而听之以心，无听之以心，而听之以气。听止于耳，心止于符。气也者，虚而待物者也。唯道集虚。虚者，心斋也。"
  summary: |
    心斋是庄子借孔子之口传授的认知准备方法，专门用于"面对强权、复杂人际环境"时的决策前心理准备。
    操作分为三个递进层次：第一步"无听之以耳"——停止依赖感官信息和外在权威；第二步"听之以心"——进入内在感受和价值判断；第三步"听之以气"——连内在判断也放下，进入完全虚空宁静的心境。
    "气"在庄子哲学中指"虚而待物"的状态——不是无知，而是"不知之知"：不主动投射欲望和成见于事物，让事物以其本然状态进入内心。庄子强调"唯道集虚"，即大道只能汇聚于虚空之地，有成见的心如同充实的容器，再也装不进新东西。
  tags: [心斋, 认知清空, 决策准备, 无为, 人间世]

- id: f05
  title: 无用之用（价值重估法）
  type: framework
  source_chapter: 内篇·逍遥游·三（惠子与庄子之辩）
  source_quote: |
    "今子有大树，患其无用，何不树之于无何有之乡、广莫之野，彷徨乎无为其侧，逍遥乎寝卧其下？不夭斤斧，物无害者，无所可用，安所困苦哉！"
  summary: |
    这是庄子著名的"无用之用"框架，用于重新评估人们日常对"有用/无用"的判断。
    惠子认为大葫芦"无用"——太大而无法盛水、无法作瓢。庄子的反驳是：你只是不知道它的用法。为什么不把它系在腰间当作腰舟浮游江湖？这里的思维跳跃是——一个东西的"用"不是固定的，而是取决于你如何定义使用场景。
    进一步地，庄子用大樗树（臭椿）说明：不被匠人看中、无法做家具的树，恰恰因为"无用"而避免了被砍伐，在旷野中自然生长千年。这说明世俗的"有用"往往是伤害性的——能盛水的容器被使用，能做家具的树被砍伐；"无用"反而是保全之道。
    决策框架意义：在做人生或商业选择时，问自己"这是谁定义的有用/无用？标准本身是否需要被质疑？"
  tags: [无用之用, 价值重估, 逆向思维, 逍遥游, 决策框架]

- id: f06
  title: 顺势而为法（庖丁解牛）
  type: framework
  source_chapter: 内篇·养生主·二（庖丁解牛）
  source_quote: |
    "臣之所好者道也，进乎技矣。始臣之解牛之时，所见无非全牛者；三年之后，未尝见全牛也；方今之时，臣以神遇而不以目视，官知止而神欲行。依乎天理，批大郤，导大窾，因其固然。"
  summary: |
    这是庄子通过庖丁解牛故事展示的决策与执行框架，适用于"在复杂约束条件下找到最优路径"的场景。
    庖丁的解牛技术分为三个境界：第一境"所见无非全牛"——眼中看到的是混乱的整体，无从下手；第二境"未尝见全牛"——三年后能够看到牛体的内部结构，有了解剖的路线图；第三境"以神遇而不以目视"——完全凭心神感应，刀锋自然顺着骨骼肌肉的缝隙游走。
    核心方法论是"因其固然"：顺应事物本来的结构和纹理，而不是用蛮力强行切割。具体操作是"批大郤，导大窾"——劈开大缝隙，引刀进入大空腔。
    转化到人生决策层面：面对复杂局面（"全牛"），不要急于行动，而是先理解其内部结构和运行规律（"天理"），找到阻力最小的路径（"有间"），以最小的强制力完成目标。
  tags: [顺势而为, 庖丁解牛, 约束条件, 决策框架, 养生主]

- id: f07
  title: 安时处顺法
  type: framework
  source_chapter: 内篇·养生主·三（秦失吊老聃）
  source_quote: |
    "适来，夫子时也；适去，夫子顺也。安时而处顺，哀乐不能入也，古者谓是帝之县解。"
  summary: |
    这是庄子通过"秦失吊老聃"故事提出的人生态度框架，专门用于应对"无法改变、无法回避"的困境。
    故事背景：老聃死了，朋友秦失前来吊丧，只哭了几声就离开了。弟子不解，秦失说那些哭得死去活来的人"遁天倍情"——逃避天然、背弃真情，因为他们没有理解生死本是一体。
    "安时处顺"的核心逻辑：人出生是"应时而生"，人死亡是"顺理而去"——两者都是大道的运行，不是人力所能左右的。既然忧乐都不能改变这个自然过程，那么"安于时、顺于命"就成为最合理的应对方式。
    庄子更进一步说，能做到"安时处顺"，则"哀乐不能入"——外界的悲喜波动无法撼动内心，达到一种情感上的绝对安宁。庄子称之为"帝之县解"——如同倒悬的人被解开束缚，恢复了自然的状态。
    这个框架的适用范围：面对生老病死、失去挚爱、事业失败等"人力不可逆转"之事，纠结、悲伤、愤怒都无济于事，正确的应对是承认现实、接纳现实、内心保持安宁。
  tags: [安时处顺, 接受, 死亡观, 决策框架, 养生主]

- id: f08
  title: 物化观照法（庄周梦蝶）
  type: framework
  source_chapter: 内篇·齐物论·六（庄周梦蝶）
  source_quote: |
    "昔者庄周梦为胡蝶，栩栩然胡蝶也。自喻适志与，不知周也。俄然觉，则蘧蘧然周也。不知周之梦为胡蝶与？胡蝶之梦为周与？周与胡蝶则必有分矣。此之谓物化。"
  summary: |
    这是庄子提出的自我边界反思框架，通过"庄周梦蝶"的著名寓言来探讨"自我认知的可靠性"问题。
    庄子在梦中化为蝴蝶，欣然自得，忘记了自己是庄周。醒来后变成庄周，不知道是庄周做梦化为蝴蝶，还是蝴蝶做梦化为庄周。庄子的结论是：两者"必有分"——终归有分别，但这分别不是绝对的。
    "物化"的含义：物与我之间的界限并非固定不变，而是可以相互转化的。这种转化在梦中发生，在极度专注时发生，在"坐忘"的精神状态中也会发生。
    对决策框架的启发：人的自我认知往往是狭隘的——我们总是以"我是XX"的方式来定义自己，但这种定义如同梦境一样是暂时的、不稳固的。在做重大人生决策时，可以暂时"悬置"自我身份，问：如果我不是我现在这个角色，我会如何选择？这有助于打破职业惯性、社会期待对判断的遮蔽。
  tags: [物化, 自我反思, 边界消解, 齐物论, 思维模型]

- id: f09
  title: 天倪和之（随是非法）
  type: framework
  source_chapter: 内篇·齐物论·五（长梧子言）
  source_quote: |
    "化声之相待，若其不相待，和之以天倪，因之以曼衍，所以穷年也。忘年忘义，振于无竟，故寓诸无竟。"
  summary: |
    这是庄子通过长梧子之口提出的应对人际冲突的实践框架。
    "化声"指是非之争的各种言论。"相待"指这些言论两两相对、相互依赖而生——你说"是"，我才说"非"；你说"非"，我才说"是"，没有一方能独立存在。"不相待"的意思是，如果能超越这种相互依赖的关系，是非之争就失去了根基。
    "和之以天倪"的具体操作：用"自然的均平"（天倪）来调和是非，让对立双方各安其位，不强求统一。进而"因之以曼衍"——顺着自然的变化而变化，不设固定目标，不做强制规划。
    最终效果是"忘年忘义"——忘记时间、忘记是非，遨游于"无竟"（无边界、无穷尽的境界）。这与"道枢"互补：道枢解决认识论问题（如何看），天倪解决实践问题（如何做）。
  tags: [天倪, 是非调和, 人际冲突, 决策框架, 齐物论]

- id: f10
  title: 浑沌之死（勿人为过度原则）
  type: framework
  source_chapter: 外篇·应帝王（寓言部分，选自相关章节）
  source_quote: |
    "日凿一窍，七日而浑沌死。"
  summary: |
    这是《应帝王》篇中"浑沌凿窍"寓言的核心警示。故事：南海之帝倏和北海之帝oto看到中央之帝浑沌没有七窍（眼耳口鼻），出于好心，每天为浑沌凿开一窍，七天后浑沌就死了。
    这个寓言提炼出的框架原则是：**人为干预有时比不干预更有害**。倏和oto的出发点是善意的（让浑沌像正常人一样感知世界），方法也是渐进的（每天一窍），但结果却是毁灭性的——因为浑沌的本性就是浑然一体，"凿窍"这个行为本身就是对浑沌本性的根本破坏。
    决策框架意义：(1)在试图改善一个系统之前，先问：这个系统的"自然状态"是什么？你的"改善"是否在破坏它的本性？(2)"好心办坏事"往往源于用人类的价值观和标准去衡量、去干预非人类系统的运行；(3)"渐进"不等于"正确"，方向错了，每一步都是错的。
  tags: [浑沌, 勿人为过度, 顺其自然, 反面教材, 决策原则]
