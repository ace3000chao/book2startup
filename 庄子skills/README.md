## 🤔 这套Skills能解决什么问题？

在人生重大决策、人际冲突、复杂困境面前，你是否曾经：

- 🤔 纠结"该不该辞职/创业"，放不下别人的看法？
- 😤 和家人/合伙人吵架，永远吵不出结果？
- 📊 看了一堆分析报告，反而更迷茫？
- 😰 面对强势的人，紧张得说不出话？
- 💼 被"有什么用"的标准绑架，不知道自己想要什么？
- 🌀 在复杂局面中处处碰壁，感觉在用蛮力？
- 😔 面对无法改变的失败，走不出来？

这些问题没有标准答案，但庄子早在2000年前就提供了独特的思考框架——**不是给你答案，而是给你看问题的新视角**。

## 📦 蒸馏了什么？

本仓库从《庄子》原文提取了 **8个可执行的方法论Skills**，每个skill都是一套完整的思考框架+可操作步骤：

| # | Skill | 你遇到这些问题时使用 |
|---|-------|---------------------|
| 1 | **无待逍遥法** | 纠结要不要做某件事，被功名/他人评价捆绑 |
| 2 | **道枢法** | 和别人争论谁对谁错，陷入二元对立 |
| 3 | **以明观物法** | 信息太多越看越迷茫，需要空掉分析直接看本质 |
| 4 | **心斋法** | 面对强权/高压紧张，脑子一片空白 |
| 5 | **无用之用** | 被"有什么用"的标准质疑，不知道自己想要什么 |
| 6 | **顺势而为法** | 复杂局面不知从何下手，处处碰壁 |
| 7 | **安时处顺法** | 面对无法改变的失败/失去，内心无法释怀 |
| 8 | **天倪和之** | 和别人吵架/立场对立，永远吵不出结果 |

## ✨ 效果示例

### 场景1：该不该辞职创业？

**用户说**："我想辞职去创业，但担心万一失败别人会怎么看我"

**调用Skill**：无待逍遥法

**AI引导**：
> 问自己一个问题：*如果做这件事，你是在依赖什么条件？*
>
> - 依赖别人的认可？
> - 依赖成功后的成就感？
> - 依赖一个"创业者"的身份标签？
>
> 如果这个条件消失了，你还会做这件事吗？
>
> 如果答案是"不会"——那这件事是"有待"的，不是真正的自由。
> 如果答案是"还会"——你已经触摸到了"无待"的门槛。

---

### 场景2：和合伙人吵架

**用户说**："我和合伙人在股权分配上各执一词，我觉得我贡献更大，他认为他资源更多"

**调用Skill**：道枢法

**AI引导**：
> 停下来。你争论的"谁贡献更大"，是在哪个层面上争论？
>
> 现在站在圆环中心看：你们两个人都在环上。争论的焦点（股权比例）在环上周而复始，没有尽头。
>
> 但你们有一个共同的目标：**公司活下去**。
>
> 站在这个"枢纽"上看问题：你们争论的不是"谁更重要"，而是"怎么能活得更久"。
>
> 股权是为了让人心甘情愿地干活，不是为了"公平"。
>
> 现在，用这个视角重新谈一次。

---

### 场景3：信息过载越看越迷茫

**用户说**："我要不要转行，看了20篇分析文章，越看越迷茫"

**调用Skill**：以明观物法

**AI引导**：
> 停。先把20篇文章放一边。
>
> 这些文章都是"小成"——每篇都是作者从自己的立场出发，局部、片面、有利益相关的。
>
> 现在闭上眼睛。不想这些文章说什么，而是问自己：
>
> *转行这件事本身——不是关于它的各种说法——它让你感受到什么？*
>
> 不是让你"想"答案，而是让答案自己浮出来。
>
> 这就是"以明"——空明的心境照见事物本然。

## ✨ 使用案例

问：我现在在课堂课教学中，发现现在05后的大学生对创新创业的兴趣度比以前低了很多，我如何让他们理解创新创业应该是一种生活态度？

以下为我的openclaw的创业导师Agent回答：

## 📚 问题本质

**05后对创新创业的认知被"窄化"了。**

他们看到的创新创业 = 开公司、融资、上市、赚钱——这是一套**功利框架**。

庄子恰恰在破这个框架。

---

## 🔍 核心分析

### 1. 《无用之用》破除功利框架

**问题：** 学生的"兴趣低"其实是因为——他们用功利尺度量创新创业，发现"我没钱/没人脉/没资源，创什么业"，然后放弃了。

**庄子智慧：** "无用之用"告诉我们：**"有用"反而是局限**。一棵树因为"不成材"反而活了千年——创新从来不是因为你"有条件"，而是来自"真正的需求和热情"。

**教学转化：** 与其讲"创业能赚钱"，不如讲"创业是解决你真实痛点的能力"——哪怕只是优化了你室友的一个小流程。

---

### 2. 《无待逍遥法》重新定义成功

**问题：** 05后的挫败感来自"比较"——看到的是马斯克、乔布斯，觉得自己不配。

**庄子智慧：** "无待"讲的是：**不依赖外在评价体系，不等"有条件了再做"**。

**教学转化：** 引导05后看到——创业不是"等风来"，而是"随时可以开始的内心状态"。用最小行动验证自己的想法，而不是等资金/团队/资源。

---

### 3. 《顺势而为法》从05后的天然优势出发

**问题：** 传统创业教育用80/90后的路径来教05后，他们找不到共鸣。

**庄子智慧：** 庖丁解牛讲的是：**顺着结构走，不硬碰**。05后是数字原住民，他们的"势"在于：

- AI工具随手可用（DeepSeek、豆包、Coze）
- 信息差几乎消失
- 表达渠道极度丰富（小红书、B站）

**教学转化：** 教他们用AI工具"顺势而为"，而不是告诉他们"你们得跟我们当年一样吃苦"。

---

### 4. 《以明观物法》回归本然

**问题：** "创业"这个词被污染了——它承载了太多社会期待、媒体叙事、成功学鸡汤。

**庄子智慧：** 以明观物法讲的是：**放下预判，让事物本然显现**。

**教学转化：** 可以用一个简单的练习——"列出你最近一个月让你不爽的一个小问题，然后想想怎么解决"。这就是创新创业的起点。让学生自己发现：**原来我每天都在做创新**。

## 💡 核心策略对比

| 传统叙事 | 庄子视角 |
|---------|---------|
| 创业 = 当老板 | 创业 = 解决你真实遇到的问题 |
| 创业需要资金/团队 | 创业 = 最小MVP验证 |
| 成功 = 上市/财富自由 | 成功 = 做你想做的事 |
| 创业是少数人的事 | 创新是每个人的自然倾向 |

## 📝 一句话总结

> **不是05后对创业没兴趣，是"创业"这个词被我们教歪了。**  
> 创新创业不是"建功立业"，而是"顺着你的生命走，做你真正想做的事"。

## 🧠 这些Skill是怎么生成出来的？

本仓库的Skills使用 [book2skill](https://github.com/cangjie-skill/book2skill) 方法论自动生成：

```
《庄子》原文
    ↓ 阶段0：整书理解（Adler四步法）
BOOK_OVERVIEW.md
    ↓ 阶段1：5个并行提取器
candidates/
    ├── frameworks.md      # 思维框架
    ├── glossary.md       # 核心术语
    ├── cases.md         # 案例故事
    └── counter-examples.md # 反例警告
    ↓ 阶段1.5：三重验证
verified.md  ←  V1跨域 + V2预测力 + V3独特性
    ↓ 阶段2：RIA++六段构造
skills/*.md  ←  R/I/A1/A2/E/B 六段结构
    ↓ 阶段3：Zettelkasten链接
INDEX.md     ←  引用关系图
    ↓ 阶段4：压力测试
skills/*.json  ←  test-prompts测试用例
```

**质量保证**：
- 每个skill必须通过"三重验证"（跨域/预测力/独特性）
- 每个skill包含边界（何时不用）的反例警告
- 全部56个测试用例，覆盖应触发/不应触发/边界场景

## 🚀 如何使用

### 方式1：在OpenClaw中使用（推荐）

（1）把skills手动安装到~/.openclaw/skills/（根据你需要使用的Agent的skills目录）

（2）和Agent对话，让它学习这个skills

（3）在日常使用中按需要与你的agent对话

如果你使用OpenClaw作为AI助手，只需说：

```
我想辞职创业，但担心别人怎么看我
```

AI会自动识别场景并调用**无待逍遥法**Skill。

或者直接指定：

```
用道枢法帮我分析一下和合伙人的矛盾
```

### 方式2：在其他AI Agent中使用

每个Skill的Markdown文件包含完整的**六段结构**：

```markdown
---
name: wu-dai-xiaoyao
description: |
  庄子的"无待逍遥"skill：当你在人生重大选择前...
  trigger信号："该不该接受这个机会""我想辞职又怕"
---

# R — Reading（原文）
> "若夫乘天地之正..."
# I — Interpretation（自述）
# A1 — Past Application（书中案例）
# A2 — Future Trigger★（何时调用）
# E — Execution（可执行步骤）
# B — Boundary（边界）
```

将描述字段（description）作为system prompt的一部分，即可让其他agent调用。

### 方式3：直接阅读学习

每个Skill都写得完整自洽，可以直接阅读作为《庄子》的现代解读：

```
skills/无待逍遥法/README.md   → 适合想理解"逍遥游"核心思想
skills/心斋法/README.md        → 适合想学习冥想/正念方法
skills/顺势而为法/README.md    → 适合想理解"庖丁解牛"的智慧
```
