# 量弘识高德行递进框架

## R — 原文 (Reading)

> "德随量进，量由识长，故欲厚其德，不可不弘其量，欲弘其量，不可不大其识。"
>
> — 洪应明, 菜根谭·量弘识高 功德日进

---

## I — 方法论骨架 (Interpretation)

**德行不够，不要直接修德——要先增见识、再弘气度。**

这个框架揭示了一个反直觉的成长路径：通常我们认为"要做一个好人，就直接去修德"（比如强制自己善良、强制自己正直）。但《菜根谭》说：这是舍本逐末。真正的路径是：**识（见识） → 量（气度） → 德（品德）**。

**三层逻辑：**
1. **识 → 量**：你的见识越广，你的气度越大。一个只见过村里的人，可能为一点小事耿耿于怀；一个见过世界的人，会知道很多事"没那么重要"。见识的扩展，自然带来气度的扩大。
2. **量 → 德**：你的气度越大，你的品德越厚。当你不再为小事计较、不再被人轻易激怒、不再患得患失，你的德行自然显现。品德不是"装"出来的，而是"自然流出"的。
3. **德是结果，不是手段**：你想厚德，但直接去修德，往往修不出来。你应该做的是：增广见识（识）、开阔气度（量），德行自然会随之厚起来。

**一句话总结：** 要想品德好，先要眼界宽；眼界要宽，先要让心大。

## A1 — 书中的应用 (Past Application)

### 案例 1: 张良的"识"
- **问题**: 张良最初是一个有勇无谋的"刺客"，在博浪沙刺杀秦始皇，失败后流亡。
- **方法论的使用**: 张良后来遇到黄石公，被传授《太公兵法》。他通过学习兵法、历史、政治，显著提升了"识"——从一个有勇无谋的刺客，变成一个有战略眼光的谋士。这个"识"的扩展，让他的"量"（气度）得以扩大——他不再为小事激动，不再为个人恩怨冲动，而是站在全局思考问题。
- **结论**: "识"的积累是第一步，有了"识"才能有"量"。
- **结果**: 张良的"大识"带来了"大量"，他的"大量"体现在：不争功、懂进退、在功成名就后全身而退。他的"德"（千古流芳的美名）是"识→量→德"路径的自然结果。

### 案例 2: 刘邦的"识"
- **问题**: 刘邦的出身比项羽低得多，初期"见识"也相对有限（一个泗水亭长）。
- **方法论的使用**: 刘邦在成长过程中不断"大其识"：他敢用韩信（突破了自己对人才的识别能力）、他听取张良的建议（扩展了自己的战略见识）、他能接受失败并调整（扩展了自己对成败的认知）。这些"识"的扩展，让他的"量"越来越大——从一个小小的亭长，到能容纳各路英雄的领袖。
- **结论**: "识"不是固定不变的，是可以通过学习和经历来扩展的。刘邦的成功在于他不断扩展自己的"识"。
- **结果**: 刘邦的"大量"让他吸引了张良、韩信、萧何等人才，最终建立汉朝。他的"德"（从一个小人物到开国皇帝）是"识→量→德"路径的体现。

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **想提升自己但找不到方向**：用户说"我想成为一个更好的人"，但不知道从哪入手，想知道"修德"的正确路径。
2. **觉得自己"知道但做不到"**：用户觉得自己"识"够了（知道什么是对的），但"量"不够（做不到），或者"量"够了但"德"不够（做不到的事情没有变成稳定的品格）。
3. **发现"做好事但没有好结果"**：用户做了很多"善意"的事，但结果不好，开始怀疑"修德有没有用"。此时需要用f17来检验：你是否跳过"识→量"的路径，直接去追求"德"？
4. **评估自己当前卡在哪个环节**：用户想知道自己"德行不够"的原因是：见识不够广（识不足）、气度不够大（量不足）、还是品德本身不够（德不足）。
5. **想设定成长路径**：用户想从A状态成长到B状态，但不知道按什么顺序。

### 语言信号 (用户的话里出现这些就应激活)

- "我想成为一个更好的人"
- "我觉得自己德行不够"
- "我要怎么提升自己"
- "我知道但做不到"
- "我做了好事但没有好结果"
- "为什么我修德没有用"
- "我觉得自己眼光不够"
- "我这个人气度太小"

### 与相邻 skill 的区分

- 与 `f15（格物致知内观框架）` 的区别：f15 是"在安静中内观自己的真心"的修炼方法；f17 是"通过扩展见识来提升气度，从而厚德"的成长路径。f15是内观，f17是外学。
- 与 `f03（动静皆静模型）` 的区别：f03 是"在任何环境下如何保持内心安定"的能力；f17是"如何通过扩展见识来提升品德"的路径。f03是状态能力，f17是成长路径。

## E — 可执行步骤 (Execution)

当 skill 被激活后， agent应按以下步骤执行:

1. **评估"识→量→德"哪个环节薄弱**
   - 完成标准: 问用户三个问题：①"我的见识是否足够广？我是否对世界有足够的了解？"（评估识）；②"我的气度是否足够大？我是否能容纳不同意见、不同个性的人？"（评估量）；③"我的德行是否稳定？我是否在没有人监督时也能做对的事？"（评估德）。
   - 判停条件: 若用户无法自我评估，则请用户举一个最近的例子——在什么情境下，你的反应体现了"识不够/量不够/德不够"？

2. **找到薄弱环节的突破口**
   - 完成标准: 根据评估结果，设定具体的突破方向：①识不足——制定"扩展见识"的具体计划（如：读什么书、去哪里、见谁）；②量不足——找一个"让你不舒服的人/事"，主动练习容纳；③德不足——找一个"你明知故犯"的问题，分析背后的动机（不是直接修德，而是找到"量不够所以德不够"的原因）。
   - 判停条件: 若三个环节都薄弱（用户状态极弱），则从"识"开始——先从最简单的"每天读30分钟书"开始。

3. **设定"识→量→德"的递进路径**
   - 完成标准: 制定一个90天的递进计划：第1个月聚焦"识"（扩展见识），第2个月聚焦"量"（练习容纳），第3个月检验"德"（是否在关键时刻能做出正确的选择）。
   - 判停条件: 若用户无法坚持（如30天后放弃了），则提示"可能设定的目标太大，从更小的开始"。

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **用户已经知道自己的问题并已经在行动**：此时"识→量→德"的框架是多余的，用户需要的不是更多方向，而是持续行动的陪伴。
- **用户的"德行不够"是借口**：有些用户说"我想修德"但实际上是在逃避具体的行动（如不想工作、不想承担责任），此时f17可能被用来合理化"不行动"。
- **用户有心理健康问题**：如果用户的"气度小"、"德行不够"实际上是抑郁症、焦虑症等心理问题的症状，则需要专业治疗，不是f17能解决的。

### 作者在书中警告的失败模式

- **器小禄薄（ce14）**：心胸狭隘、念头迫促，斤斤计较于眼前小利。f17指出：气度小（量不足）的人，德行自然也不会厚。ce14正是f17要帮助用户识别和改变的状态。
- **标节义道学之患（ce12）**：刻意标榜自己的节义和道学，以善名自居——这是"跳过识→量直接修德"的极端形式。标榜节义者必为节义受毁谤，因为这种"德行"是表演出来的，不是从识→量的路径中自然流出的。

### 容易混淆的邻近方法论

- **"识→量→德" vs "直接修德"**：直接修德是"我要善良、要正直"，但往往停留在意志力层面，无法持久。f17指出：真正的德行是从见识和气度中自然流出的，不是有意为之。
- **"厚德" vs "表演德行"**：厚德是"因为见识广、气度大，所以自然展现出好的品格"；表演德行是"因为我想让别人觉得我好，所以刻意表现善良"。前者是自然流出，后者是刻意表演。
