# 骄泰失之谦虚受益原则

## R — 原文 (Reading)

> "富贵家宜宽厚而反忌刻，是富贵而贫贱其行矣，如何能享？聪明人宜敛藏而反炫耀，是聪明而愚懵其病矣，如何不败？"
>
> — 洪应明, 《菜根谭·谦虚受益 满盈招损》

---

## I — 方法论骨架 (Interpretation)

1. **富贵家应该宽厚，但很多人反而刻薄。** 有钱有权后对别人吝啬计较，行为和穷人没有区别——这叫"富贵而贫贱其行"，怎么可能享福？
2. **聪明人应该敛藏，但很多人反而炫耀。** 有点本事就到处显摆，这是"聪明而愚懵其病"——真正的愚蠢。
3. **骄傲带来两个问题：行为上刻薄（对别人），认知上愚昧（对自己）。**
4. **"富贵宜宽厚"的逻辑：财富是用来帮助别人的，不是用来彰显自己的。** 宽厚待人，才能保住财富；刻薄待人，财富会流失。
5. **"聪明宜敛藏"的逻辑：炫耀聪明会招来嫉妒和攻击，藏起来才能安全。** 麦穗弯腰是常识，聪明人也该如此。

---

## A1 — 书中的应用 (Past Application)

### 案例: 项羽和李斯的败亡——富贵忌刻

- **问题**: 为什么有些富贵之家很快败落？
- **方法论的使用**: 洪应明指出项羽初期强大却因刚愎自用、刻薄寡恩而败于刘邦；李斯位极人臣却因不知收敛而身死族灭
- **结论**: 富贵之家如果刻薄待人，行为和贫贱者无异，怎么可能保住富贵？
- **结果**: 项羽乌江自刎，李斯被腰斩；刘邦因宽厚待人而成帝业

### 案例 2: 聪明人炫耀卖弄的下场

- **问题**: 为什么有些聪明人反而失败？
- **方法论的使用**: 洪应明指出聪明人如果炫耀卖弄，就是"聪明而愚懵其病"——看似聪明实则愚蠢
- **结论**: 真正有智慧的人如麦穗般谦逊低头，越炫耀自己越显得缺知少识
- **结果**: 炫耀聪明的人，最终都招来嫉妒、攻击和失败

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要 this skill?

1. **成功/得意后，感到自己很了不起时** — "我很厉害"、"他们都得听我的"、"我应该得到更多"
2. **取得成绩后，想要炫耀/展示时** — "我想让他们知道我做到了"、"我应该让人知道我多厉害"
3. **对不如自己的人态度变差时** — "他们不如我，有什么资格和我说话？"、"他算什么？"
4. **觉得自己掌握了什么，想要显摆时** — "你不知道我知道什么"、"这你都不懂？"

### 语言信号 (用户的话里出现这些就应激活)

- "我很厉害"
- "他们都不如我"
- "我应该得到更多"
- "我想让他们知道我做到了"
- "他算什么？"
- "这你都不懂？"

### 与相邻 skill 的区分

- 与 `p21毋偏信自任毋自满嫉人` 的区别: p21是"日常中防止四种陷阱（偏信/自任/傲短/嫉能）"，p27是"成功后防止骄傲"；p21是持续的，p27是时机性的
- 与 `p03心地光明才华韫藏` 的区别: p03是"才华要藏"，p27是"成功后要收敛"；p03是日常原则，p27是成功后的应对

## E — 可执行步骤 (Execution)

当 skill 被激活后，agent 应按以下步骤执行:

1. **确认用户处于"成功/得意/富贵/聪明"的状态**
   - 完成标准: 用户明确自己处于上升期或成功后

2. **用"富贵而贫贱其行"测试：你的行为和你的地位匹配吗？**
   - 完成标准: 用户检查自己是否在得意时对别人刻薄

3. **用"聪明而愚懵其病"测试：你是在炫耀还是真的强？**
   - 完成标准: 用户意识到炫耀聪明的害处

4. **"敛藏"测试：如果现在把你的成就藏起来，你会损失什么？**
   - 完成标准: 用户发现"显摆"没有任何实质收益，只有风险

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **建立权威需要的场合**：领导需要展示能力来建立威信，过度"谦逊"会失去领导力
- **激励团队需要的场合**：适度展示成就可以激励团队，不是所有炫耀都是坏的
- **需要建立个人品牌的场景**：现代职场需要适度展示，过度收敛可能影响发展

### 作者在书中警告的失败模式

- 骄泰失之（来自本原则）：富贵后对不如自己的人态度变差，取得成绩后到处炫耀，对他人吝啬计较
- 机制：富贵宜宽厚却反忌刻，聪明宜敛藏却反炫耀；两个错误都源于骄傲

### 作者的盲点 / 时代局限

- 现代职场强调"个人品牌"，适度展示是必要的
- "敛藏"在某些文化中是美德，在另一些文化中可能是缺乏自信的表现

### 容易混淆的邻近方法论

- **心地光明才华韫藏(p03)**：p03是"才华要藏"，p27是"成功后要收敛"；p03是日常原则，p27是成功后的应对
- **无过是功(p10)**：p10强调"不邀功"，p27强调"成功后收敛"；p10是功利性谦逊，p27是成长性谦逊
