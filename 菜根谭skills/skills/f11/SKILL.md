# 舍己勿疑施恩勿报框架

## R — 原文 (Reading)

> "舍己毋处其疑，处其疑，即所舍之志多愧矣；施人毋责其报，责其报，并所施之心俱非矣。"
>
> — 洪应明, 菜根谭·舍己勿疑 施恩勿报

---

## I — 方法论骨架 (Interpretation)

**做真正的利他行为——事前不犹豫，事后不求报。**

这个框架提供了两条自我检验标准：
1. **舍己时：不要犹豫计较**——如果你在"舍"的时候还在想"值不值"、"亏不亏"，那么这个"舍"的纯粹性已经打了折扣。真正的舍己是"舍了就是舍了"，不是"舍了等着回报"。
2. **施恩时：不要期待回报**——如果你帮助别人是为了"有一天让他还"，那么这个"施恩"的行为就已经变味了。真正的施恩是"我帮你，因为我选择帮你"，不是"我帮你，因为你欠我"。

**方法论本质：**
- 判断一个行为是不是"真利他"，核心标准是："事后会不会后悔？"
- 如果舍的时候犹豫，说明你还没准备好舍——可以先想清楚，再决定舍还是不舍。
- 如果施的时候期待回报，说明这个施的本质是投资——不是不可以，但不要骗自己这是"纯粹利他"。

**一句话总结：** 舍己时问"舍了之后会不会后悔"——不会后悔的舍，才是真舍；施恩时问"如果对方永远不还，我还帮吗"——会的帮，才是真帮。

## A1 — 书中的应用 (Past Application)

### 案例 1: 大禹治水三过家门而不入
- **问题**: 大禹受命治水，面对的是一个需要30年专注投入的巨大工程，这意味着在父亲死后（父亲因治水失败被处死）继续承担风险，以及长期离开家人。
- **方法论的使用**: 大禹没有"处其疑"——他没有在"要不要舍这个家"的问题上犹豫计较。他选择了全力以赴治水，三过家门而不入，这个"舍"是纯粹的。"舍己"的志没有打折。
- **结论**: 不犹豫的舍，才是真正的舍。大禹的舍己行为成为千古楷模，正是因为他的"舍"没有犹豫、没有计较、没有期待回报。
- **结果**: 治水成功，大禹建立夏朝，成为"圣人"。他的成功来自于"舍己时的不犹豫"——这种纯粹性让他能够全力以赴。

### 案例 2: 刘邦的"散财"与项羽的"吝啬"
- **问题**: 刘邦和项羽在争夺天下时，都需要争取人才。项羽是"吝啬"的（舍不得封赏），刘邦是"大方"的（舍得封赏）。
- **方法论的使用**: 从"施恩勿报"的角度看，刘邦的"施"不是期待某一个人回报，而是"我用封赏换取你的忠诚和效力"——这是一种选择，不是一种投资。如果刘邦"责其报"（要求每个人必须立功才能封赏），他的施恩就变味了，人才也会流失。
- **结论**: "施恩勿报"不是说不能有目的地施惠，而是说施恩时不要期待即时回报，更不要在施恩后念念不忘。
- **结果**: 刘邦以"舍得施"汇聚了韩信、张良、萧何等人才，最终建立汉朝；项羽以"吝啬"失去范增等人才，最终乌江自刎。

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **决定是否为他人牺牲利益**：用户面临"是否要为了帮助别人而放弃自己的利益"的选择，不确定自己该不该舍、舍了会不会后悔。
2. **评估利他行为的动机**：用户帮了别人，但开始计较"我帮了他，他怎么没有感谢我"——想知道自己的动机是否纯粹。
3. **合作关系中"谁付出更多"的失衡**：用户觉得自己付出很多但没有得到相应的回报，想知道是自己计较还是对方真的不公平。
4. **决定是否要为一个机会放弃现有利益**：用户面临"是否要放弃现在稳定的收入去追求一个不确定的机会"——需要检验这个决定的牺牲是否纯粹。
5. **评估自己的"舍己"是否值得**：用户已经做了一次"舍"，但开始怀疑这个"舍"是否值得，需要一个框架来检验。

### 语言信号 (用户的话里出现这些就应激活)

- "我该不该为他牺牲"
- "我付出了这么多值得吗"
- "我对别人好但没有得到回报"
- "我帮了他，他却没有谢我"
- "我在想要不要放弃这个去追那个"
- "我觉得我付出比对方多"
- "我舍了这么多，但好像没有好结果"
- "我在计算值不值"

### 与相邻 skill 的区分

- 与 `f09（天理人欲分判框架）` 的区别：f09 问的是"这条路是公利还是私利"（方向问题）；f11 问的是"我的牺牲是纯粹的还是计量的"（动机问题）。f09是方向，f11是动机。
- 与 `f01（退让一步决策框架）` 的区别：f01 问"要不要退让"（策略问题）；f11 问"我的舍是否纯粹"（动机问题）。f01是战术，f11是道德检验。

## E — 可执行步骤 (Execution)

当 skill 被激活后， agent应按以下步骤执行:

1. **确认"舍"或"施"的具体内容**
   - 完成标准: 明确说出"你舍了什么？或你施了什么？"，并量化（如：放弃了多少收入、付出了多少时间、让渡了多少权益）。
   - 判停条件: 若用户说不清楚自己舍/施了什么（只是模糊的"付出"感），则要求其量化后再继续。

2. **检验"舍时的动机纯度"**
   - 完成标准: 问用户两个问题：①"舍的时候你在犹豫吗？"（如果犹豫，说明动机不纯）；②"舍的时候你在计算回报吗？"（如果有计算，说明动机不纯）。
   - 判停条件: 若两个问题都是"是"（犹豫且计算），则提示"你可能还没准备好舍，建议重新考虑"。
   - 若两个问题都是"否"（不犹豫不计算），说明这是真正的"舍己"。
   - 若是一个"是"一个"否"，进入下一步的"事后测试"。

3. **"事后测试"——问自己会不会后悔**
   - 完成标准: 让用户想象"这件事结束之后，我会后悔吗？"如果会后悔，说明这个"舍"不是真舍——需要重新决定；如果不会后悔，说明这是真正的"舍己"。
   - 判停条件: 若用户无法判断"会不会后悔"，则说"给自己24小时冷静期，再做决定"。

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **对方不值得你舍**：如果对方是"以怨报德"的人，或者是一个会利用你的牺牲的人，此时"舍己"可能变成"助纣为虐"。需要先判断对方是否值得。
- **舍了之后你会陷入生存危机**：如果"舍"会让你失去基本的生活保障，这不是"舍己"，这是"自残"。"舍己"的边界是：你舍了之后还能好好活。
- **施恩会让你变成被操控的"好人"**：如果你帮了某人会让你陷入被对方长期操控的关系，这不是"施恩"，这是"自我牺牲的强迫症"。

### 作者在书中警告的失败模式

- **以短攻短以顽济顽（ce15）**：当众揭人短处、以愤怒厌恶对待别人的愚顽，恶性循环。如果你的"施恩"变成了"我帮了你，你必须感恩"，这正是以短攻短的模式——你帮了对方，但期待对方的回报，否则就心生怨恨。
- **伪善诈善之患（ce16）**：伪装善良的伪君子比真小人更危险。表现为表面一套背后一套、满嘴道德实则自私、为自己的堕落找借口。f11如果变成"表演性的舍己"或"表演性的施恩"，就会滑向ce16的伪善。

### 容易混淆的邻近方法论

- **"舍己勿疑" vs "冲动牺牲"**：舍己是经过思考后的选择；冲动牺牲是一时情绪下的决定。f11要求的是"想清楚再舍"，不是"一时冲动就舍"。
- **"施恩勿报" vs "没有底线地帮人"**：施恩勿报是"帮了人不期待回报"；没有底线地帮人是"无论对方是否值得都帮"。前者是纯粹，后者是软弱。f11要求你"选择帮谁"——帮是你的选择，不是对方的权利。
