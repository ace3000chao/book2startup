# 以失意之思制得意之念框架

## R — 原文 (Reading)

> "自老视少，可以消奔驰角逐之心；自瘁视荣，可以绝纷华靡丽之念。"
>
> — 洪应明, 以失意之思 制得意之念

---

## I — 方法论骨架 (Interpretation)

这是**时间透视法**——用站在人生终点或未来某个节点的"你"，来回看当下的执念，从而稀释执念的强度。

两种透视角度：

1. **以老视少**（从老年回看青年）：你现在拼命追逐的东西，十年/二十年后还重要吗？站在80岁的自己看今天，还会为这件事如此焦虑吗？

2. **自瘁视荣**（从衰败回看荣耀）：你现在拥有的风光，如果已经失去，还值得你这么患得患失吗？

核心逻辑：执念之所以强烈，是因为它占据了全部视野。引入时间维度后，同样的东西会显得渺小——不是否认它的存在，而是给它一个合适的比例感。

## A1 — 书中的应用 (Past Application)

### 案例 1: 曹操人生如朝露
- **问题**: 曹操在征战生涯中，面对权势荣耀的诱惑，担忧自己是否迷失在追逐中
- **方法论的使用**: 曹操用"譬如朝露，去日苦多"的视角，提醒自己人生短暂，荣耀只是瞬间
- **结论**: 既然人生苦短，就应该珍惜时间做真正有意义的事，而不是被虚荣牵绊
- **结果**: 曹操成为一代枭雄，但始终保持对人生短暂的清醒认知

### 案例 2: 陈子昂登幽州台的感慨
- **问题**: 陈子昂怀才不遇，面对壮志难酬的困境，执着于功名
- **方法论的使用**: 他用"前不见古人，后不见来者，念天地之悠悠"的时空无限视角，看清个人功名的渺小
- **结论**: 个人的荣辱在宇宙面前不过是一粒尘埃
- **结果**: 陈子昂虽然政治上不得志，但成为唐代诗歌革新的重要人物

### 案例 3: 长城万里今犹在（反例警示）
- **问题**: 秦始皇追求万世功名，修长城、立霸业
- **方法论的使用**: 若用"以老视少"的框架检验：万里长城至今犹在，但秦始皇早已化为尘土
- **结论**: 追求身后名也是执念的一种表现
- **结果**: 追求永恒的权势，最终都无法对抗时间的消解

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **成功高潮期**：项目上线大获成功、获得融资、升职加薪，整个人处于极度兴奋中，开始规划更大的扩张
2. **沉没成本执念**：已经投入大量时间/金钱/感情在一个方向上，明知不对但不愿放弃（"我都干了这么久……"）
3. **患得患失期**：在得与失之间极度焦虑，特别是在重要决策前夜反复权衡
4. **对某个人/关系极度执着**：无法放下某段关系或某个人，反复纠缠

### 语言信号 (用户的话里出现这些就应激活)

- "我太想成功了"
- "这次机会太好了，不能错过"
- "我已经投入这么多了，不能放弃"
- "等我升职了/有钱了就轻松了"
- "我觉得这次一定能成"
- "我不甘心"（不甘心通常就是得意之念的变体）
- "为什么我总是得不到"（与"以失意之思"对应，过于执着于"得"）

### 与相邻 skill 的区分

- 与 `f10 苦中有乐 得意生悲` 的区别：f10是预判福祸转化的推理框架（苦→乐/乐→悲），本框架是操作性的时间透视技术
- 与 `f06 宠辱不惊情绪决策框架` 的区别：f06强调"当下"的情绪管理（花开花落/云卷云舒），本框架强调"跨时间"的视角切换

## E — 可执行步骤 (Execution)

当 skill 被激活后, agent 应按以下步骤执行:

1. **确认执念对象**
   - 完成标准: 用户明确说出"我现在最放不下的是什么"——要具体到一件事，而不是"我放不下很多事"

2. **执行时间透视（两选一）**
   - 完成标准 A（以老视少）：用户想象自己80岁，回看今天这件事，问自己"那时候我还会care吗？"
   - 完成标准 B（自瘁视荣）：用户想象自己已经失去了现在拥有的一切（工作/关系/财富），问自己"那时的我会怎么看今天的执念？"
   - 判停条件: 若用户目前处于严重低谷（刚失败/刚失恋），则跳过"自瘁视荣"，只用"以老视少"（引导看向未来的希望，而非放大当下的失去）

3. **写出"稀释后的判断"**
   - 完成标准: 用户用一句话写下时间透视后的结论，例如"这件事在10年后看，根本不重要"或"失去这份工作，我仍然可以……"

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 用户正处于严重失败或抑郁状态（刚被裁/刚分手），这会放大"失意之思"的负面影响，让他们更加自我否定——这时候应该用f13逆增上缘
- 需要立即行动的关键时刻（如谈判中途、危机处理中），时间透视会让人犹豫不决
- 刚取得合法合理成功的人（正常的高兴不应该被稀释）

### 作者在书中警告的失败模式

- ce03（快意不知回首）："恩里由来生害，故快意时须早回首"——得意时不知退让是最大的危险，但本框架的"以老视少"不能变成自我否定，要保持中性
- ce17（倚物役心）：用时间透视来强化"什么都不重要"的虚无感，而不是用来校准执念强度——这是误用

### 容易混淆的邻近方法论

- **f23（来之去之循环转化框架）**：也是时空视角，但f23更极端（微尘/泡影），是消解性视角；本框架是校准性视角，保留执念但给它合适比例
