# 格物致知内观框架

## R — 原文 (Reading)

> "夜深人静，独坐观心，始知妄穷而真独露，每于此中得大机趣；既觉真现而妄难逃，又于此中得大惭忸。"
>
> — 洪应明, 菜根谭·静中观心 真妄毕见

---

## I — 方法论骨架 (Interpretation)

**定期内观，让"真"浮出来，让"妄"穷尽。**

这个框架提供了一个定期"自我检修"的程序。它的核心发现是：人在日常的忙碌和社交中，内心被各种"妄念"（杂念、欲望、防御、伪装）占据，你很难看到自己真实的想法。只有在"夜深人静，独坐观心"这种独处状态下，妄念逐渐"穷尽"（耗尽、消散），真心才会"独露"（显现出来）。

**方法论步骤：**
1. **创造独处的条件**：找一个不被打扰的时间和空间，关掉手机、远离人群，让外界刺激降到最低。
2. **观察内心起了什么念头**：不要评判，不要引导，只是"看着"内心起了什么念头、什么情绪、什么担忧、什么期待。
3. **识别"妄穷真露"的转折点**：当内心足够安静时，杂念会逐渐减少，你会感到一种"清明"的时刻——这就是真心显现的时候。
4. **面对真心，承担责任**：真心显现时，往往会伴随着"大惭忸"——意识到自己平时的虚伪、自欺、逃避。"知妄"是改变的起点，"真现"是方向的确立。

**一句话总结：** 不是外求，而是内观。在安静中，让真的显现，让假的无处躲藏。

## A1 — 书中的应用 (Past Application)

### 案例 1: 诸葛亮的"草堂春睡"
- **问题**: 诸葛亮在出山前，在草堂中躬耕陇亩、静待时机。他并非只是"等待"，而是在独处中不断内观——审视自己的志向、能力、时局。
- **方法论的使用**: 诸葛亮在"夜深人静"的独处中"观心"，知道自己的志向是"兼济天下"，知道自己的才能足以辅助明主，知道当时的时局需要他出山。"妄穷真独露"——在对天下大势的反复思考中，杂念（隐居的安逸、逃避乱世的选择）逐渐消散，真心（辅助刘备、匡扶汉室）逐渐显现。
- **结论**: 独处内观是确认真心的过程，不是逃避现实的手段。
- **结果**: 诸葛亮27岁出山，辅佐刘备，此后鞠躬尽瘁、死而后已。他的选择不是冲动的决定，而是在长期内观后确认的真心。

### 案例 2: 王阳明的"龙场悟道"
- **问题**: 王阳明被贬至贵州龙场——一个瘴气弥漫、官员流放之地。他处于人生的最低谷，没有权力、没有资源、远离文明。
- **方法论的使用**: 王阳明在极端困境中，没有放弃内观。他在龙场的草屋里"静坐"，日复一日地"观心"。在这个与世隔绝的环境中，往日的杂念（对官位的执念、对个人安危的担忧、对不公正待遇的愤怒）逐渐穷尽，他开始触及更深层的问题：什么是圣人之道？我的心是什么？
- **结论**: "妄穷真独露"——在最孤独的环境中，外在的一切标签和防御被剥夺，真心才有机会显现。
- **结果**: 王阳明在龙场悟道，提出"知行合一"、"致良知"的学说，成为明代心学的创立者。他的悟道不是书斋中的空想，而是在孤独的内观中实现的。

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **定期自我复盘**：用户在年度/季度/月度复盘时，想找一个内观的方法来审视自己，而不是简单地总结"做了什么事"。
2. **在高压后需要消化期**：用户刚经历了一个高压项目/事件，想找一个安静的方式来"消化"这段时间发生的事。
3. **需要做重大决定但无法判断**：用户面临一个重大选择（职业转型、关系终止、创业决定），需要知道自己的真心是什么，但日常的忙碌和杂念让他无法看清。
4. **感觉迷失了自己**：用户感到"我不知道自己要什么"、"我不知道自己真正在乎什么"，想通过内观来重新连接自己。
5. **想识别自己的"妄念"**：用户觉得自己经常被各种杂念干扰，想知道"我真正想要的是什么"和"什么是干扰我的假象"。

### 语言信号 (用户的话里出现这些就应激活)

- "我需要静一静"
- "我想独处一下"
- "我独处时才能想清楚"
- "最近事情太多了，我需要消化一下"
- "我想知道自己到底要什么"
- "我好像迷失了"
- "我需要一个安静的时间"
- "我需要做决定，但静不下来"

### 与相邻 skill 的区分

- 与 `f08（推己及人对治换位框架）` 的区别：f08 是"对他人不满时如何换位理解"（向外看）；f15 是"在独处中内观自己的真心"（向内看）。f08是关系修复，f15是自我认知。
- 与 `f12（一念转魔成佛框架）` 的区别：f12 是"情绪即将失控时如何转念"的即时干预；f15 是"在安静中审视自己的内心"的长期修炼。f12是危机处理，f15是日常修行。
- 与 `f03（动静皆静模型）` 的区别：f03 是"在任何环境下如何保持内心安定"（测试内心是否平静）；f15 是"创造条件让真心显现"（主动创造安静来内观）。f03是状态测试，f15是过程修炼。

## E — 可执行步骤 (Execution)

当 skill 被激活后， agent应按以下步骤执行:

1. **创造独处的条件**
   - 完成标准: 确定一个不被打扰的时间和空间（至少30分钟），明确说"在这段时间里，我会关掉手机、不看消息、不与人交谈"。
   - 判停条件: 若用户无法找到独处的时间（太忙、责任太多），则提示"即使5分钟的安静也好，重要的是创造这个空间"。

2. **"观心"——观察内心起了什么**
   - 完成标准: 在安静中，开始观察内心起了什么念头。不要引导，不要评判，只是"看着"。记录下来：出现了哪些念头、情绪、担忧、期待、欲望。
   - 判停条件: 若用户无法安静（焦虑到无法坐下来），则提示"先做深呼吸（5次），再开始"。

3. **识别"妄穷真露"的转折点**
   - 完成标准: 继续观察，直到出现一个"安静"/"清明"的时刻——杂念减少，感到一种内在的清晰。在这个时刻，问自己："此刻我知道的最真实的一件事是什么？"
   - 完成标准: 记录这个"真心"，并写下"我平时在哪些情况下会忽略/压抑这个真心？"
   - 判停条件: 若30分钟内没有出现"清明"状态，则结束今天的内观，记录"今天未能触及真心，可能是时机未到"。

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **用户正在经历急性心理危机**：如严重抑郁发作、有自我伤害冲动。此时内观可能让用户更深入地接触痛苦的感受，需要专业支持而非独自内观。
- **用户在创伤后应激状态（PTSD）**：独处和安静可能触发闪回或其他创伤反应，不宜在没有专业支持下进行内观。
- **用户无法区分"真心"和"强烈情绪"**：有时我们把"强烈的情绪"当成"真心"（比如愤怒、恐惧），但实际上这些可能不是真正想要的，而是创伤反应。

### 作者在书中警告的失败模式

- **机械深沉之患（ce02）**：饱经世故后变得世故圆滑、工于心计，恰恰是在"观心"时用技巧代替真诚——用"我应该如何想"来代替"我真正在想什么"，这是假内观，不是真内观。
- **标节义道学之患（ce12）**：刻意标榜自己的节义和道学，以善名自居——在内观时把自己的念头"编辑"成道德上可接受的版本，而不是真实的版本。"知妄"的前提是"真实地知道"，如果在内观时自我欺骗，则永远无法"妄穷真露"。

### 容易混淆的邻近方法论

- **"内观" vs "反省自责"**：内观是"不带评判地看"，反省自责是"带着审判地看"。前者是接纳，后者是批判。f15要求的是"观心"不是"审判心"。
- **"内观" vs "胡思乱想"**：内观是"观察已经存在的念头"；胡思乱想是"主动制造念头来填充空白"。前者是被动的观察，后者是主动的逃避。f15要求的是"观察"，不是"创造"。
