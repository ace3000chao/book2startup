# 净从秽生转化框架

## R — 原文 (Reading)

> "粪虫至秽，变为蝉而饮露于秋风；腐草无光，化为萤而跃彩于夏月。因知洁常自污出，明每从晦生也。"
>
> — 洪应明, 菜根谭·净从秽生 明从暗出

---

## I — 方法论骨架 (Interpretation)

**污秽不是终点，是转化的原材料。**

这个框架的核心洞见是：洁净与污秽、光明与黑暗不是对立的两极，而是可以互相转化的条件性存在。粪虫在污秽中生长，却能化为蝉，在秋风中饮露；腐草在无光中沉睡，却能化为萤火虫，在夏夜跃彩。逆境（污秽、黑暗）不是人生的死胡同，而是某种转化的必经阶段。

**方法论本质：**
1. **接受"此时此地"的污秽**：不要急着逃离或否认现状，因为逃离本身就是在拒绝转化。
2. **寻找转化条件**：转化需要时间、态度、行动——不是"立即变干净"，而是"在污秽中积累转化所需的能量"。
3. **相信转化的可能性**：污秽→洁净、黑暗→光明，是需要时间的自然过程，不能急，但也不能放弃。

**一句话总结：** 此刻的污秽，可能是未来纯净的养料。重要的不是"我现在有多脏"，而是"我是否在创造转化的条件"。

## A1 — 书中的应用 (Past Application)

### 案例 1: 孟母三迁
- **问题**: 孟子和母亲最初居住在墓地附近，孟子学的是丧葬跪拜；搬到市场旁，孟子学的是买卖争吵。环境对人的影响极为负面。
- **方法论的使用**: 孟母没有抱怨"这地方太糟糕了"，也没有"逆来顺受"认命，而是主动识别"污秽"的环境（墓地、市场）不适宜孩子成长，主动寻求"转化条件"——搬到学堂附近，让孟子接触到好的环境和人群。
- **结论**: 洁净不是逃避污秽，而是在污秽中寻找创造转化条件的路径。
- **结果**: 孟子成为儒家亚圣，"孟母三迁"成为环境转化、个人成长的经典案例。

### 案例 2: 姜子牙钓鱼愿者上钩
- **问题**: 姜子牙年迈，空有一身才华，却在渭水之滨垂钓为生，看似处于人生最低谷——无钱无势无名，是"腐草无光"的处境。
- **方法论的使用**: 姜子牙没有因为处境卑微而放弃，而是以"愿者上钩"的姿态等待时机——他把低谷期当成积累和等待的阶段，不断修炼自己的能力，等待周文王这个"转化条件"出现。
- **结论**: 在人生的"晦"处（无光期），不是要逃离，而是要持续积累，等待转化条件的成熟。
- **结果**: 姜子牙被周文王发现，辅佐周朝八百年基业，成为"太公望"。腐草化为萤火虫的故事，在他身上真实上演。

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **处于人生低谷**：失业、失恋、创业失败、考试落榜，觉得"这辈子就这样了"，看不到希望。
2. **环境无法改变**：暂时无法换工作、换城市、离开一段有毒的关系，但又必须在这个环境中继续待着。
3. **感觉"太脏了不可能有好结果"**：用户在污秽/混乱的环境中感到绝望，认为这个环境决定了命运，看不到任何正向可能性。
4. **想逃离但无处可去**：用户在污秽的现状中挣扎，但既无法改变现状也无法离开，不知道该怎么办。
5. **做决定前评估转化可能性**：用户面临是否要"放弃现在重新开始"的决策，需要一个判断框架来决定是转化还是逃离。

### 语言信号 (用户的话里出现这些就应激活)

- "我觉得现在太绝望了"
- "这个环境太脏了，还能有什么好"
- "我这辈子就这样了吗"
- "现在的处境太糟糕了，看不到希望"
- "我想离开但走不了"
- "这里的人都这样，我也不可能例外"
- "这些经历会不会毁了我"
- "什么时候才能熬出头"

### 与相邻 skill 的区分

- 与 `f07（见微知著守正待时）` 的区别：f07 讲"时机不到要等"，强调的是"时机"；f05 讲"污秽中可以转化"，强调的是"条件"。f07 更偏时机等待，f05 更偏主动创造转化。
- 与 `f13（逆增上缘修行框架）` 的区别：f13 讲"逆境是修行的机会"（接受逆境），f05 讲"逆境中可以转化为净"（改造逆境）。f13 是认知框架，f05 是转化路径。

## E — 可执行步骤 (Execution)

当 skill 被激活后，agent 应按以下步骤执行:

1. **确认当前处境的"污秽度"**
   - 完成标准: 明确说出"你现在处于什么状态"（低谷/困境/有毒环境），并用"1-10"评估这个状态的恶劣程度（10是最恶劣）。
   - 判停条件: 若用户评估<3，说明并非真正的"污秽"处境，此框架适用性低，应转介更轻量的skill。

2. **寻找"转化微生物"**
   - 完成标准: 找出"在这个污秽环境中，有什么可以成为转化的材料？"——包括：①在这个环境中能否学习到某些技能；②是否有可利用的资源；③是否有正向的人际关系可以建立。
   - 判停条件: 若找不到任何可转化的因素，且用户已处于极端困境（如严重心理创伤），则提示寻求专业支持。

3. **设定"转化时间表"**
   - 完成标准: 和用户一起设定一个"转化里程碑"——不是"立即变干净"，而是"在3-6个月后，我在这个环境中的状态要有X改变"。
   - 判停条件: 若用户设定的时间表过于激进（如"一个月内必须完全改变"），则提示调整预期，说明转化需要时间。

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **用户处于严重的心理/身体危险中**：如被虐待、严重抑郁、有自杀倾向，此时最优先的是安全脱离危险，而不是在"污秽中找转化"。
- **污秽是结构性问题且无法改变**：如用户处于极度贫困、严重歧视、无法获得基本生存资料的环境中，此时"净从秽生"可能是对现实的逃避，应先帮助用户认识真实处境。
- **用户需要立即的行动方案**：此框架是心态调整工具，不提供具体行动方案。如果用户需要"告诉我现在该做什么"，应转介其他skill。

### 作者在书中警告的失败模式

- **飞蛾扑火（ce18）**：明明有更好的选择（晴空万里）却偏要扑向危险（灯火）。表现为：明知某些路是死路，却偏偏执意行之；贪恋危险的关系或机会；重复同样的错误。f05的滥用可能变成"合理化继续待在有害环境中"的理由，需警惕。
- **鸱鸮嗜腐鼠（ce18）**：有清泉绿果可以饮食，却偏偏喜欢吃腐烂不堪的死鼠。表现为：为了一己私利而不知舍弃不义之物，明知有害却执迷不悟。

### 容易混淆的邻近方法论

- **"净从秽生" vs "美化苦难"**：净从秽生是承认苦难存在，并寻找其中的转化可能性；美化苦难是假装"苦难是好事"而拒绝承认其伤害。前者是转化思维，后者是自我欺骗。
- **"净从秽生" vs "逆来顺受"**：净从秽生是主动寻找转化条件；逆来顺受是放弃改变现状，接受所有的不公。f05要求行动，不是被动接受。
