# 自适其性回归框架

## R — 原文 (Reading)

> "峨冠大带之士，一旦睹轻蓑小笠飘飘然逸也，未必不动其咨嗟；长筵广席之豪，一旦遇疏帘净几悠悠焉静也，未必不增其绻恋。"
>
> — 洪应明, 生活自适其性 贵人不若平民

---

## I — 方法论骨架 (Interpretation)

这是**需求真伪识别法**——通过反差刺激，发现你现在追求的东西未必是你真正需要的。

核心洞察：我们往往不知道自己真正想要什么，但我们知道自己不想要什么。通过观察"我们被什么所吸引"，可以反推"我们真正需要什么"。

两个反向刺激：

1. **达官贵人看到平民的自由飘逸，未必不动心**：那些拥有名利的人，看到简朴自由的生活，内心往往涌起向往——说明自由比名利更接近真实需求。

2. **富豪看到简朴生活，未必不向往**：奢华之人在遇到"疏帘净几"的清净环境时，往往感到眷恋——说明清净比奢华更接近真实需求。

推论：**你现在追求的东西，可能只是别人告诉你应该追求的，而不是你真正需要的。** 检验方法：想象你已经拥有了它，然后观察你是否仍然感到不满足。

## A1 — 书中的应用 (Past Application)

### 案例 1: 陶渊明归隐田园（反向验证）
- **问题**: 陶渊明在官场中拥有一定的地位和俸禄，但他感到不满足
- **方法论的使用**: 他用"反差刺激"发现：自己在官场中的"峨冠大带"并不快乐，而看到"轻蓑小笠"的农人生活，内心反而向往
- **结论**: 他真正需要的不是官位，而是自由和自然
- **结果**: 陶渊明辞官归隐，写出了"采菊东篱下，悠然见南山"——中国隐逸文化的开端

### 案例 2: 刘禹锡陋室铭
- **问题**: 刘禹锡被贬官，失去地位和财富
- **方法论的使用**: 他发现"长筵广席"的奢华并没有带来内心的安宁，而"陋室"的简朴反而让他保持自在
- **结论**: 安宁不在于物质的丰富，而在于内心的充实
- **结果**: 《陋室铭》流传千古，刘禹锡成为中国文学史上的标杆人物

### 案例 3: 范蠡功成身退
- **问题**: 范蠡辅佐越王勾践灭吴，拥有了极大的权力和名誉
- **方法论的使用**: 他用反差刺激发现：权力巅峰的同时也意味着风险（"敌国破，谋臣亡"），而泛舟五湖的自由才是真正的幸福
- **结论**: 权力不是真正需要的，自由和生命才是
- **结果**: 范蠡功成身退，后经商成为"陶朱公"，善终而富

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **职业选择迷茫**：在多个Offer之间纠结，或者不知道该继续打工还是创业，或者要不要转行
2. **生活方向迷茫**：已经有了一定的成就或物质条件，但总觉得"不是真正想要的"
3. **消费决策困境**：想买某件贵重物品（房/车/奢侈品），但不确定是真正需要还是社会比较
4. **关系选择迷茫**：在多段关系之间纠结，或者不确定现在的关系是否真的适合自己

### 语言信号 (用户的话里出现这些就应激活)

- "我不知道自己想要什么"
- "总觉得不是真正想要的"
- "我应该过什么样的生活"
- "选哪个都觉得不对"
- "我现在追求的真的是我需要的吗"
- "为什么我有了这些还是不开心"
- "如果我有钱了/升职了/结婚了就……"（填完后感到不确定）
- "好像少点什么，但说不上来是什么"

### 与相邻 skill 的区分

- 与 `f20 以失意之思制得意之念` 的区别：f20是用时间透视稀释执念（未来视角），本框架是用反差发现真实需求（对比视角）
- 与 `f15 格物致知内观框架` 的区别：f15是"静下来往内看"，本框架是"通过对比刺激往内看"——一个是独处，一个是借外境

## E — 可执行步骤 (Execution)

当 skill 被激活后, agent 应按以下步骤执行:

1. **列出"现在拥有的"和"向往的"**
   - 完成标准: 用户分别写下两项清单：
     a. "我现在拥有的"（职业/收入/关系/社会地位等）
     b. "我内心真正向往的"（不需要理由，直觉判断）

2. **寻找"反差刺激"**
   - 完成标准: 用户回忆或想象一个场景：在那个场景中，你看到了完全不同生活方式的人，你内心感到了一丝触动——那个触动是什么？
   - 判停条件: 若用户找不到任何触动（对什么都无感），可能是情感隔离，需要先处理情感问题

3. **问"拥有了还想要吗"**
   - 完成标准: 用户对清单a中的每一项，问自己："如果我已经拥有了它，而且永远不会再失去，我还想要什么？"——第一个冒出来的答案，往往更接近真实需求

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 基本生存需求未满足的人（缺钱/缺安全感）——他们需要的是解决温饱，不是探索真实需求
- 把"向往简朴"当成逃避责任的借口（不想工作所以"向往田园"）——这是合理化成瘾，不是真正的自我探索
- 严重抑郁症患者——他们可能无法区分"真实需求"和"症状性需求"

### 作者在书中警告的失败模式

- ce12（标节义道学之患）："标道学者，常因道学招尤"——若把"自适其性"当成一种新的标签和追求（"我要活出真我"），则又陷入新的执念
- ce17（倚物役心）：若用这个框架得出的结论是"我什么都不需要"（虚无主义），而不是"我真正需要的是简单和自由"，则是误用

### 容易混淆的邻近方法论

- **f20（以失意之思）**：都是"对比"，但f20是对比"不同时间的自己"，本框架是对比"自己与社会建构的期待"
