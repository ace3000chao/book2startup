# 圣凡一体观

## R — 原文 (Reading)

> "标节义者，必以节义受谤；标道学者，常因道学招尤。故君子不近恶事，亦不立善名，只浑然和气，才是居身之珍。"
>
> — 洪应明, 浑然和气 处事珍宝

---

## I — 方法论骨架 (Interpretation)

这是**不走极端的中道处世框架**，针对的是"道德洁癖"和"虚伪善名"两种极端：

两个极端的危险：

1. **标榜节义（道德洁癖）**：越标榜自己的道德，反而越容易因道德受诽谤。因为一旦有了"我是好人"的人设，任何小失误都会被放大。

2. **标榜道学（虚伪善名）**：越强调自己的道学修养，越容易招来非议。因为"强调道学"本身就已经不道学了。

正确的态度：**不近恶事**（不做坏事）+ **不立善名**（不追求"好人"标签）= **浑然和气**

核心逻辑：真正的高手，不做好事以证明自己是好人，也不做坏事以迎合世俗。他们只是自然地存在着，不走两个极端。这就是"圣凡一体"——既不圣洁到脱离世俗，也不凡俗到失去自我。

## A1 — 书中的应用 (Past Application)

### 案例 1: 项羽和李斯败亡的对比
- **问题**: 项羽拥有极高天赋和贵族身份，却因极端性格而败亡
- **方法论的使用**: 项羽"富贵而反忌刻"——有富贵却刻薄待人，不走中道；李斯聪明却"以物役我"，不知收敛。两者都走了极端
- **结论**: 极端的性格（不论是极善还是极恶）都会导向失败
- **结果**: 项羽乌江自刎，李斯被腰斩；反观刘邦以宽厚待人而成帝业

### 案例 2: 包拯和海瑞的平衡
- **问题**: 包拯和海瑞以刚正著称，但刚正到极点会有问题
- **方法论的使用**: 两人虽然严格执法，但并不是为了"立善名"，而是自然地做该做的事；他们也懂得在必要时妥协
- **结论**: 刚正本身没问题，但若"标榜刚正"就会变成"偏激"
- **结果**: 包拯、海瑞千古流芳，但不是以"完人"形象，而是以"和而不同"的方式被后人记住

### 案例 3: 曾国藩的修身之道
- **问题**: 曾国藩位极人臣，功高震主，如何保全自身
- **方法论的使用**: 他深知"标节义者必受谤"的道理，在功成名就后主动裁军，不立功名，以"浑然和气"自处
- **结论**: 不立善名，才能保全自己
- **结果**: 曾国藩功成身退，善终于世

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **道德拉扯**：在"坚持原则"和"灵活变通"之间纠结，不确定什么时候该坚持、什么时候该妥协
2. **形象焦虑**：担心自己"太清高"被人疏远，或者担心"太随和"被人利用
3. **标签执念**：执着于"我是好人/有原则的人/不好说话的人"等标签，被标签所累
4. **人际平衡困惑**：不知道对不同的人应该用什么态度——对上级要谄媚吗？对下级要严厉吗？

### 语言信号 (用户的话里出现这些就应激活)

- "我不知道该坚持原则还是随大流"
- "我怕自己太清高了"
- "做好人会吃亏怎么办"
- "既要做好人又要不吃亏，可能吗"
- "我是不是太较真了"
- "我总觉得要保持形象，不能……"
- "别人都这么做，我要不要也……"（跟随vs坚持的拉扯）
- "我不知道该怎么拿捏这个度"

### 与相邻 skill 的区分

- 与 `f24 不为法缠不为人缠框架` 的区别：f24强调"不被任何规范束缚"，本框架强调"不刻意立善名也不刻意做恶"——两者都走中道，但f24更激进（彻底不缠），本框架更日常（不标榜）
- 与 `f04 中和处世决策框架` 的区别：f04关注性格偏激（躁/刻/执），本框架关注道德形象（不立善名）

## E — 可执行步骤 (Execution)

当 skill 被激活后, agent 应按以下步骤执行:

1. **识别"标签执念"**
   - 完成标准: 用户问自己："我是不是执着于'我是XX的人'这个标签？"——找到自己最执着的一个标签

2. **执行"去标签测试"**
   - 完成标准: 用户想象：如果我明天不是"XX的人"了，我还是我吗？——如果答案是"是"，则标签是工具；如果答案是"否"，则标签是牢笼
   - 判停条件: 若用户处于需要明确道德立场的场合（如商业谈判/法律事务），先去标签化可能会导致失去立场，此时需结合f09判断

3. **找到"浑然和气"的基准**
   - 完成标准: 用户回答：什么行为是"不刻意做善事，但也不做恶事"的具体表现？（例：不给别人贴标签、不主动炫耀自己的付出、不因为"我是好人"而委屈自己）

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 需要明确道德立场的场合（如商业欺诈/违法行为）——此时应该坚持立场，不应该用"不立善名"作为借口逃避
- 抑郁症患者觉得自己"不够好"——这会让他们更加自我否定
- 把"不立善名"当成"不做善事"的借口

### 作者在书中警告的失败模式

- ce12（标节义道学之患）："标节义者，必以节义受谤"——刻意立善名本身就是问题所在
- ce16（伪善诈善之患）："君子而诈善，无异小人之肆恶"——若把"不立善名"当成伪装，继续暗中作恶，则比明目张胆的恶人更有害

### 容易混淆的邻近方法论

- **f24（不为法缠不为人缠）**：两者都强调"不缠"，但f24强调不被任何东西束缚（包括"善"本身），本框架强调"善"要自然不刻意
