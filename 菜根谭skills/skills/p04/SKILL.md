# 快意早回头原则

## R — 原文 (Reading)

> "恩里由来生害，故快意时须早回首；败后或反成功，故拂心处莫便放手。"
>
> — 洪应明, 《菜根谭·快意早回头 拂心莫放手》

---

## I — 方法论骨架 (Interpretation)

1. **顺境中藏着危险。** "恩里由来生害"——恩情深重时往往是祸患降临之时，得意忘形之际最容易被反噬。
2. **顺境时要懂得"回头"。** 快意时不是继续进攻，而是要急流勇退，见好就收。
3. **逆境中可能藏着机会。** 失败不等于终结，"败后或反成功"——挫折有时反而是通往成功的必经之路。
4. **逆境时要保持在场。** 拂心（心里不痛快）时不要轻易放弃，因为放弃可能错过了即将到来的转机。
5. **核心是对称原则：顺境收，逆境撑。** 不是永远进攻，也不是永远撤退，而是根据处境调整。

## A1 — 书中的应用 (Past Application)

### 案例: 范蠡、张良功成身退 vs 李斯、吴王刘濞贪权身败名裂

- **问题**: 如何判断什么时候该退，什么时候该进
- **方法论的使用**: 洪应明列举两组人物——功成身退者（范蠡、张良）与贪恋富贵者（李斯、吴王刘濞），用对比说明"知退"的重要性
- **结论**: 范蠡辅佐越王勾践灭吴后功成身退，泛舟五湖成为巨富；张良辅佐刘邦建立汉朝后选择隐退修仙——二人都得以善终。李斯辅佐秦始皇统一六国后因贪恋权势被赵高陷害，具五刑而死；吴王刘濞因谋反兵败被杀
- **结果**: "快意时不知回首"的人身败名裂；"功成身退"者反而留名青史

### 案例 2: 谢安淝水之战后退隐

- **问题**: 事业鼎盛时期是否该继续追逐更大权力
- **方法论的使用**: 谢安在淝水之战立下大功、功名鼎盛之时，急流勇退，不贪恋权位
- **结论**: 在最辉煌的时候选择退隐，才能全身而退、终老山林、名垂青史
- **结果**: 谢安善终，与那些不知退让而败亡的人物形成鲜明对比

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **事业/人生处于高峰期，感到志得意满时** — "我好不容易到这个位置，不能退"、"这么好的时候不继续太可惜了"
2. **面对更大诱惑（更高职位、更大权力、更多金钱）时** — "这个机会太难得了，不抓住会不会后悔？"
3. **看到身边的人都在追涨（投资、扩张、冒进）时** — "我是不是太保守了？"、"别人都在进我退是不是傻？"

### 语言信号 (用户的话里出现这些就应激活)

- "我现在很顺，是不是该收一收？"
- "这个机会太好了，放弃可惜"
- "我还能再进一步吗？"
- "别人都在追，我该不该跟？"
- "我已经到这个位置了，放手太难"

### 与相邻 skill 的区分

- 与 `p11退让一步加让三分` 的区别: p11是日常人际中的退让策略；p04是针对个人事业巅峰期的战略撤退
- 与 `p28守正安分远祸` 的区别: p28是"守正"——守住正道避开祸患；p04是主动选择"急流勇退"

## E — 可执行步骤 (Execution)

当 skill 被激活后，agent 应按以下步骤执行:

1. **判断用户当前是"快意"还是"拂心"状态**
   - 完成标准: 用户明确自己处于顺境得意还是逆境困境

2. **若是"快意"（顺境）：用"恩里生害"检测——这个顺境有没有隐藏的危险？**
   - 完成标准: 用户识别出恩情/成功背后可能酝酿的祸患（如树大招风、功高震主）

3. **"早回首"测试：如果现在这个状态再持续3年，最坏的可能性是什么？**
   - 完成标准: 用户意识到继续冒进的潜在风险，并考虑退出的时机和方式

4. **若是"拂心"（逆境）：用"败后反成功"检测——这个挫折是不是通往某个机会的必经之路？**
   - 完成标准: 用户重新定义失败的意义，找到坚持的理由

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **战略进攻期**：企业/个人正在扩张期，需要一鼓作气拿下市场份额时
- **低谷期的关键坚持**：创业濒临失败时，如果放弃是正确的选择（不是所有坚持都有意义）
- **家庭/健康危机时**：此时需要的是止损而非硬撑

### 作者在书中警告的失败模式

- 快意不知回首：处于巅峰时愈发贪婪，不愿接受任何退让，享受众人吹捧不能自拔
- 机制：恩深往往生害，盛极必致衰败。得意时不知退让，树大招风，祸患随至

### 作者的盲点 / 时代局限

- 古代官场逻辑（功成身退）与现代商业逻辑（持续增长）不完全一致
- 某些行业需要"All in"才能成功，过早退出可能错过历史性机会

### 容易混淆的邻近方法论

- **退让一步(p11)**：p11是人际互动中的退让；p04是事业巅峰期的战略撤退
- **守正安分(p28)**：p28是被动防御；p04是主动选择进退时机
