# 断绝思虑逍遥框架

## R — 原文 (Reading)

> "斗室中，万虑都捐，说甚画栋飞云、珠帘卷雨；三杯后，一真自得，唯知素琴横月、短笛吟风。"
>
> — 洪应明, 断绝思虑 光风霁月

---

## I — 方法论骨架 (Interpretation)

这是**极简减法操作法**——不是增加什么来获得自由，而是**减少什么**来获得自由。

核心公式：**简到极致 = 真境**

三个递减层次：

1. **万虑都捐**：把一切忧虑、挂碍、算计全部清除。不是选择性放下，是全部捐出。
2. **环境降到最低**：不需要画栋飞云的豪宅，斗室（小屋）就够了。
3. **欲望降到最低**：不需要锦衣玉食，三杯淡酒、一把琴、一支笛就够了。

推论：真正的快乐和自由，不需要复杂的条件。条件越复杂，离真越远。这与"成功需要很多资源"的常识相反——真正的逍遥，是有勇气和能力过简单生活的人才能达到的境界。

## A1 — 书中的应用 (Past Application)

### 案例 1: 刘禹锡《陋室铭》
- **问题**: 刘禹锡被贬官，居于破旧陋室，物质条件极差
- **方法论的使用**: 他没有追求更好的物质条件，而是以"斯是陋室，惟吾德馨"自处——物质简陋但精神充实
- **结论**: 环境简陋不等于内心简陋，心正则境不陋
- **结果**: 《陋室铭》成为千古名篇，"陋室"成为高洁志趣的象征

### 案例 2: 陶渊明桃花源记
- **问题**: 陶渊明身处乱世，仕途不得志，对现实政治失望
- **方法论的使用**: 他不是去改造现实或谋取更高的官职，而是彻底"断绝思虑"——归隐田园，过最简单的农耕生活
- **结论**: "桃源犬吠，桑间鸡鸣"——最淳朴的生活才是真境
- **结果**: 陶渊明成为中国隐逸文化的精神偶像，其诗歌开创了田园诗派

### 案例 3: 姜子牙钓鱼愿者上钩
- **问题**: 姜子牙空有一身才华，却长期无人赏识，处境困顿
- **方法论的使用**: 他没有拼命钻营或自我推销，而是"以静制动"——在江边钓鱼，静待时机
- **结论**: 不需要主动追逐，最简单的存在方式即可
- **结果**: 最终遇到周文王，成就一番功业

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **思维反刍循环**：对某件事反复思考停不下来（睡前/凌晨/休息时），严重影响睡眠和情绪
2. **条件依赖症**：总觉得"等我有了XX条件就能开始了"——买了设备/租了办公室/请了人，但实际上行动力仍然为零
3. **选择瘫痪**：面对太多选项（职业选择/投资决策/人生方向），反而无法行动
4. **精神消耗过度**：总觉得累，但说不出具体在累什么；无法真正放松

### 语言信号 (用户的话里出现这些就应激活)

- "我脑子停不下来，一直在想同一件事"
- "想太多做不下去"
- "怎么才能真正放下"
- "等我有了XX就能……"（条件依赖句式）
- "我需要好好想想"（但已经想了很久）
- "每天脑子里装太多东西了"
- "感觉被压得喘不过气"
- "我现在状态不对，需要调整"

### 与相邻 skill 的区分

- 与 `f19 忙里偷闲 闹中取静` 的区别：f19是在忙乱中保持节奏（资源管理），本框架是彻底清空（减法革命）
- 与 `f12 一念转魔成佛框架` 的区别：f12是在情绪腾沸的瞬间"转念"，本框架是系统性的"万虑都捐"

## E — 可执行步骤 (Execution)

当 skill 被激活后, agent 应按以下步骤执行:

1. **识别"条件清单"**
   - 完成标准: 用户列出"我认为做成X事需要的所有条件"，然后逐条问自己："如果没有这条，我真的做不成吗？"——大多数条件答案是"其实没有也行"

2. **执行"减法三问"**
   - 完成标准: 用户回答三个问题：
     a. "我现在最不需要担心的一件事是什么？"（挑最大的那个放下）
     b. "我一直在想的那件事，1年后还重要吗？"（如果否，现在为什么要想？）
     c. "如果我现在一无所有，只能留一件事，留什么？"（找到真正的核心）
   - 判停条件: 若用户处于真实匮乏状态（基本生存需求未满足），则先满足基本需求，不要强行"减法"

3. **设定"最小可执行单元"**
   - 完成标准: 基于步骤2的发现，用户确定一个"不需要任何条件，现在就能开始"的最小的行动

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 用户正处于真实危机或紧急情况（需要行动，而不是放下）——应该用`f33 不动声色应变框架`
- 用户的基本生活需求未得到满足（缺钱/缺安全感），强行"万虑都捐"是自欺欺人
- 需要深度思考的创造性工作（写方案/策划/学习）——此时不应减法，而应用f19管理节奏
- 抑郁症患者过度思考——需要专业干预，不是减法能解决的

### 作者在书中警告的失败模式

- ce02（机械深沉之患）："历事深，机械亦深"——把"断绝思虑"当成城府深沉的伪装，而不是真正的放下
- ce17（倚物役心）："以物役我者，逆固生憎，顺亦生爱"——若表面上减少物质，实际上仍然被其他东西（评价/名声/关系）所役，则没有真正执行这个框架

### 容易混淆的邻近方法论

- **f19（忙里偷闲）**：f19是资源管理（提前存款），本框架是彻底清空。f19适合日常高压，本框架适合思维过载
- **f12（一念转魔成佛）**：f12是瞬间的"转念"技术，本框架是系统性的"减法"操作
