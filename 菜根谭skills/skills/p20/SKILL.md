# 放下屠刀原则

## R — 原文 (Reading)

> "当怒火欲水正腾沸处，明明知得，又明明犯着，知的是谁，犯的又是谁？此处能猛然转念，邪魔便为真君矣。"
>
> — 洪应明, 《菜根谭·放下屠刀 立地成佛》

---

## I — 方法论骨架 (Interpretation)

1. **情绪沸腾时，人知道自己在做什么，但控制不住。** "明明知得，又明明犯着"——理智知道不对，身体和情绪却不听使唤。
2. **"知的是谁，犯的又是谁？"——这是洪应明的点睛之笔。** 理智是"知"，情绪是"犯"；它们是两个不同的"我"，不是同一个。
3. **关键动作：猛然转念。** 不是慢慢冷静，不是一再忍耐，是在一个瞬间——转念。
4. **"邪魔便为真君"——一念之间，凡人成圣。** 转念前是恶魔，转念后是君子；差别就在这一念。
5. **不是压抑情绪，是转换焦点。** 从"我要报复"转换成"这样做的后果是什么"或者其他更高视角。

---

## A1 — 书中的应用 (Past Application)

### 案例: 洪应明论"猛然转念，邪魔便为真君"

- **问题**: 当怒火欲水沸腾时，如何不被情绪控制？
- **方法论的使用**: 洪应明指出"知"和"犯"是两个不同的主体，转念就是在它们之间创造一个空间
- **结论**: 一念之间，从恶魔变成君子；差别不在环境，在这一念的转换
- **结果**: 在最冲动的时候，能够"猛然转念"的人，避免了多少悲剧

---

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要 this skill?

1. **极度愤怒，想做出伤害性行为时** — "我要让他付出代价"、"我不会让他好过的"
2. **被仇恨/报复心占据，想要报复时** — "我恨死他了"、"他必须受到惩罚"
3. **被欲望冲昏头脑，想要做出越界行为时** — "我知道不该，但我控制不住"
4. **情绪失控，做出事后后悔的事之前** — "我忍不住了"、"我现在就要做"

### 语言信号 (用户的话里出现这些就应激活)

- "我控制不住自己了"
- "我现在就要做"
- "我要让他付出代价"
- "我恨死他了"
- "我知道不该，但我停不下来"
- "我忍不住了"

### 与相邻 skill 的区分

- 与 `p13中和为福偏激为灾` 的区别: p13是长期性格调整（避免偏激），p20是即时情绪控制（怒火时猛然转念）；p13是战略，p20是战术
- 与 `p14谨言慎行` 的区别: p14是日常谨言慎行，p20是极端情绪时的紧急制动；p14是日常驾驶，p20是紧急刹车

## E — 可执行步骤 (Execution)

当 skill 被激活后，agent 应按以下步骤执行:

1. **确认用户正处于"怒火欲水正腾沸"的极端情绪中**
   - 完成标准: 用户明确自己情绪极度激动

2. **用"知的是谁，犯的又是谁"提醒用户：你的"理智"和"情绪"是不同的声音**
   - 完成标准: 用户意识到有另一个声音在看着自己冲动

3. **引导"猛然转念"——让用户从"我要报复"切换到另一个视角**
   - 完成标准: 用户能够说出"如果我做了，后果是什么？"或者"一年后看这件事，我还会在意吗？"

---

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **用户处于真正的危险中需要自卫**：此时需要战斗反应，不是"放下屠刀"
- **严重心理健康危机**：如重度抑郁/躁怒发作，可能需要专业医疗介入
- **对方正在造成客观伤害需要制止**：需要实际行动保护自己和他人，不是单纯心态调整

### 作者在书中警告的失败模式

- 飞蛾扑火（来自反例ce18）：明知危险却偏扑灯火，为名利所诱自取灭亡
- 机制：执迷不悟，一意孤行，直到铸成大错

### 作者的盲点 / 时代局限

- 某些情绪（如被侵犯后的愤怒）是正当的，需要被认可而不是简单"放下"
- "猛然转念"需要内心修炼的积累，没有一定修为的人很难做到

### 容易混淆的邻近方法论

- **中和为福(p13)**：p13是避免偏激，p20是情绪失控时的紧急制动；p13是预防，p20是治疗
- **不恶小人(p12)**：p12是事后对小人"不恶"，p20是情绪当时的"猛然转念"；p12是结果，p20是过程
