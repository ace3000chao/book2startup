# 循序三修框架

## R — 原文 (Reading)

> "德随量进，量由识长，故欲厚其德，不可不弘其量，欲弘其量，不可不大其识。"
>
> — 洪应明, 量弘识高 功德日进

---

## I — 方法论骨架 (Interpretation)

这是**自我修养的递进路径框架**，核心洞察是：**品德不能直接修，要通过"先增见识、再阔气量、最后厚德"的迂回路径来达成。**

三层递进结构：

1. **识（见识）**：先要增广见识——见多识广，才能知道世界有多大、自己有多小。见识少的人，气量必然狭窄。

2. **量（气度）**：见识广了，气量自然变大——因为见过不同的活法，就不会再为小事斤斤计较。气量是见识的自然结果。

3. **德（品德）**：气量大了，品德自然厚了——一个有见识、有气量的人，不需要刻意"修德"，自然会有德。

核心误区：大多数人想"直接修德"（从0到德），但没有见识和气量支撑的"德"，是脆弱的、表面的、表演性的。正确的路径是：识 → 量 → 德。

## A1 — 书中的应用 (Past Application)

### 案例 1: 大禹治水——见识先行
- **问题**: 大禹之前，鲧用"堵"的方法治水失败了
- **方法论的使用**: 大禹通过实地考察（增广见识），发现水宜疏不宜堵——这就是"识"，是后来一切的基础
- **结论**: 没有见识，就没有正确的判断；没有正确的判断，再努力也是徒劳
- **结果**: 治水成功，建立夏朝

### 案例 2: 姜子牙的超级见识
- **问题**: 姜子牙拥有极高的才能，但始终没有施展的机会
- **方法论的使用**: 他没有因此放弃或抱怨，而是用等待的时间继续增广见识和研究——直到见识到了足以支撑大业的程度
- **结论**: 见识够了，气量自然大，机会自然来
- **结果**: 佐周灭纣，成就大业

### 案例 3: 刘邦vs项羽的气量对比
- **问题**: 刘邦出身低微，项羽出身贵族，为什么刘邦最终得了天下？
- **方法论的使用**: 项羽"见识"虽高（兵法世家），但气量小（不能容人）；刘邦见识不如项羽，但气量大（能容人用人）——从气量对比看，刘邦胜
- **结论**: 项羽的直接"修德"（恢复贵族秩序）是空洞的；刘邦的"容人雅量"是见识和气量结合的结果
- **结果**: 汉朝建立，刘邦成开国皇帝

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **自我反省但找不到入口**：觉得自己"格局不够/气量太小/修养不足"，想要改善但不知从哪下手
2. **直接"修德"失败**：尝试过各种方法修身养性，但总是坚持不下来或没有效果
3. **想帮助他人成长**：作为管理者/导师，想知道如何帮助下属或学生提升
4. **评估自己处于哪个阶段**：不确定自己是"见识不够"还是"气量不够"还是"品德不够"

### 语言信号 (用户的话里出现这些就应激活)

- "我想提高自己的修养"
- "怎么才能有更大的格局"
- "我觉得自己气量太小了"
- "要怎样才能厚德"
- "我知道我应该大度，但我做不到"
- "道理我都懂，但还是……"（识→量的转化出了问题）
- "读了很多书，但还是过不好"（识有了但量不够）
- "我的问题到底在哪里"（需要诊断框架）

### 与相邻 skill 的区分

- 与 `f17 量弘识高德行递进框架`（f31的姐妹框架，同一段原文的不同角度解读）的区别：f17强调"识→量→德"的递进逻辑，本框架强调"不要直接修德，要从根源入手"
- 与 `f08 推己及人对治换位框架` 的区别：f08是具体的换位操作，本框架是宏观的修养路径诊断

## E — 可执行步骤 (Execution)

当 skill 被激活后, agent 应按以下步骤执行:

1. **诊断当前阶段**
   - 完成标准: 用户回答三个问题，确定自己处于哪一阶段：
     a. "我见过的世界够大吗？"（如果视野狭窄，则处于"识"阶段）
     b. "我能容纳不同意见吗？"（如果不能，则处于"量"阶段）
     c. "我自然而然有德吗？"（如果不自然，则处于"德"阶段）

2. **针对阶段设定行动**
   - 完成标准: 若是"识"阶段 → 制定"见识扩张计划"（旅行/读书/见人/换环境）；若是"量"阶段 → 制定"气量训练计划"（主动接受不同意见/学会延迟反应）；若是"德"阶段 → 则无需刻意修德，保持自然
   - 判停条件: 若用户存在基本的人格障碍（自恋型/反社会型），此框架不适用，需要专业干预

3. **设定递进里程碑**
   - 完成标准: 用户写出三个月后的目标：具体是"见识多了什么"或"气量大了什么"，而不是抽象的"我要更有修养"

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 把"增广见识"当成逃避行动的借口（"我还要多看看世界才能决定"）
- 已经处于紧急情况（危机决策）——这时候不是修德的时候
- 把"修德"当成道德表演（f30已经警告过）——本框架是为了真正提升，不是为了立人设

### 作者在书中警告的失败模式

- ce02（机械深沉之患）："历事深，机械亦深"——把增广见识变成了"学套路"，而不是真正开阔心胸
- ce12（标节义道学之患）：若把"修德"变成"学道德表演"，则识越多、量越大，反而成了更高级的虚伪

### 容易混淆的邻近方法论

- **f08（推己及人）**：都是自我提升工具，但f08是具体操作（换位思考），本框架是宏观路径（识→量→德）
