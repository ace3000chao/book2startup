# Skill: 杂于利害框架

## R — 原文引用
> "是故智者之虑，必杂于利害。杂于利而务可信也，杂于害而患可解也。"
> ——孙子·九变第八

## I — 方法论骨架

**孙子的辩证智慧：**
- 单纯看到"利" → 盲目乐观 → 忽视风险
- 单纯看到"害" → 消极保守 → 错失机会
- **杂于利害** → 既看到利又看到害 → 理性决策

**两个转化公式：**
```
利 → 害的转化
"杂于利而务可信也"
= 在顺境中想到潜在危险 → 风险可控

害 → 利的转化
"杂于害而患可解也"
= 在困境中看到转化可能 → 找到出路
```

**创业者常见认知偏误：**
| 偏误 | 表现 | 后果 |
|------|------|------|
| 只看利 | "这个市场太大了！" | 忽视竞争、烧钱过快 |
| 只看害 | "风险太大了..." | 永远不行动 |
| 利无害 | "我们不一样" | 对风险视而不见 |
| 害无利 | "这行没法做了" | 放弃本可突破的领域 |

## A1 — 书中案例

九变第八中"屈诸侯者以害"：
- 用"害处"迫使对手就范
- 看似"有害"的威胁，转化为了己方的"利"

**商业案例：** 华为"备胎计划"
- "实体清单"是巨大的"害"
- 但"备胎转正"让华为技术独立
- 害 → 逼出了真正的实力

## A2 — 未来触发情境
- 🎯 看到"暴富"机会，内心激动时
- 🎯 团队士气低落，觉得"没希望"时
- 🎯 政策突变，行业面临整顿时
- 🎯 竞争对手获得大额融资，自己缺钱时
- 🎯 产品推广困难，用户不认可时

## E — 执行步骤

### 步骤1：画"利弊对照表"
| 维度 | 利 | 害 |
|------|---|---| 
| 金钱 | | |
| 时间 | | |
| 声誉 | | |
| 关系 | | |
| 能力 | | |

### 步骤2：找"利中隐藏的害"
问："如果这件事失败了，最可能的原因是什么？"

### 步骤3：找"害中隐藏的利"
问："这件事最坏的结果，能倒逼出什么机会？"

### 步骤4：决策
- 利 > 害，且害可控 → 行动
- 害 > 利，或害不可控 → 谨慎
- 利 = 害 → 寻找新方案

## B — 边界（什么时候不适用）

❌ **边界1：** 需要快速决断时
- 过度分析会错过时机
- 适用于重大决策，不适用于日常事务

❌ **边界2：** 信息极度匮乏时
- 此时分析是"垃圾进，垃圾出"
- 行动 > 分析

❌ **边界3：** 情绪极端时
- 人在极度恐惧或贪婪时，分析失真
- 先调整情绪，再分析
