# 《孙子兵法》整书理解

## 基本信息
- **书名：** 孙子兵法
- **作者：** 孙武（春秋）
- **出版年：** 2019年（刘建立译注，华中科技大学出版社）
- **字数：** 约6000字
- **地位：** 世界第一部兵书，"兵学圣典"

## 核心主旨
以"慎战"为底色，以"知彼知己"为方法论，以"不战而屈人之兵"为最高境界，构建了一套完整的战争哲学与竞争策略体系。

## 五大核心概念

1. **道** - 令民与上同意（人心向背）
2. **知彼知己** - 贯穿全书的方法论核心
3. **奇正相生** - 战术变化的根本法则
4. **致人而不致于人** - 掌握主动权
5. **合于利而动** - 理性决策准则

## 五大方法论框架

### 1. 五事七计框架（始计）
道、天、地、将、法 + 主孰有道/将孰有能/天地孰得/法令孰行/兵众孰强/士卒孰练/赏罚孰明

### 2. 诡道十二法（始计）
能而示之不能、用而示之不用、近而示之远、远而示之近、利而诱之、乱而取之、实而备之、强而避之、怒而挠之、卑而骄之、佚而劳之、亲而离之、攻其无备、出其不意

### 3. 谋攻四层楼（谋攻）
上兵伐谋 → 其次伐交 → 其次伐兵 → 其下攻城

### 4. 九变五利（九变）
途有所不由、军有所不击、城有所不攻、地有所不争、君命有所不受（因地制宜）

### 5. 五间俱起（用间）
因间、内间、反间、死间、生间

## 批判性分析

### 孙子的局限性/盲点
1. **轻视后勤与经济基础** - 过度强调速战，对持久战的经济基础论述不足
2. **地形分类过于简略** - 缺乏现代地图概念
3. **对士兵主动性论述有限** - "愚士卒"思想
4. **缺乏海战/空战思维** - 仅限陆地战争
5. **道义论述薄弱** - 纯粹功利主义

### 商战应用的注意事项
1. 商战≠兵战：商业竞争可以合作共生，兵战是你死我活
2. 诡道有边界：商业信誉是长期资产
3. 速战不总是最优：市场培育需要时间
