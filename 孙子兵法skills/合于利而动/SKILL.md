# Skill: 合于利而动框架

## R — 原文引用
> "合于利而动，不合于利而止。"
> ——孙子·九地第十一、火攻第十二

## I — 方法论骨架

**决策的唯一标准：利**

孙子在两处强调这一原则：
- 九地第十一："合于利而动，不合于利而止"
- 火攻第十二："非利不动，非得不用，非危不战"

**三层决策过滤器：**

| 层级 | 问题 | 判断标准 |
|------|------|---------|
| 利 | 这个行动对我有利吗？ | 利益 > 成本 |
| 得 | 我有把握做成吗？ | 能力圈内的机会 |
| 危 | 不行动的代价是什么？ | 机会成本分析 |

**被排除的错误决策驱动：**
- ❌ 愤怒驱动："他欺负我，我必须报复"
- ❌ 面子驱动："大家都做了，我不做没面子"
- ❌ 恐惧驱动："不做的话，别人会怎么看我"
- ❌ 贪婪驱动："这个机会看起来太诱人了"

## A1 — 书中案例

火攻第十二中孙武的警告：
> "主不可以怒而兴师，将不可以愠而致战。怒可以复喜，愠可以复说，亡国不可以复存，死者不可以复生。"

**翻译：** 君主不能因为愤怒而发动战争，将领不能因为怨恨而出战。愤怒可以变成高兴，怨恨可以变成喜悦，但国家灭亡就不能复兴，人死了就不能复生。

**反面案例：** 吴王夫差伐齐
- 表面"有利"（吞并齐国）
- 实际隐患（忽略越王勾践的威胁）
- 最终"有害"（亡国）

## A2 — 未来触发情境
- 🎯 被竞品刺激，想要"跟进"时
- 🎯 团队提出"战略性亏损"扩张计划时
- 🎯 朋友邀请"一起干大事"时
- 🎯 政策红利出现，想"追风口"时
- 🎯 情绪低落，想通过"做大事"振奋团队时

## E — 执行步骤

### 步骤1：列出行动的"利"与"害"
| | 利 | 害 |
|---|---|---|
| 立即 | | |
| 短期 | | |
| 长期 | | |

### 步骤2：计算"净利"
```
净利 = Σ(短期利 + 长期利) - Σ(短期害 + 长期害)
```
- 净利 > 0 → 考虑行动
- 净利 < 0 → 禁止行动
- 净利接近0 → 谨慎行动

### 步骤3：设置"止损线"
如果失败，最坏结果是什么？
- 能承受 → 行动
- 不能承受 → 寻找降低风险的方案再行动
- 无论如何不能承受 → 放弃

## B — 边界（什么时候不适用）

❌ **边界1：** 战略布局期
- 有些"不利"是战略性投入（研发、品牌）
- 需要用更长远的视角判断"利"

❌ **边界2：** 生死存亡时刻
- 当不行动的代价是"死"时
- 再"不利"的行动也要做

❌ **边界3：** 信任/名誉无法用钱衡量时
- 某些承诺是"无价"的
- "利"不仅是金钱，还包括信任积累
