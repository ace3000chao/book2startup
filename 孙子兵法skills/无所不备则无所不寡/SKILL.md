# Skill: 无所不备则无所不寡

## R — 原文引用
> "故备前则后寡，备后则前寡，备左则右寡，备右则左寡，无所不备，则无所不寡。寡者，备人者也；众者，使人备己者也。"
> ——孙子·虚实第六

## I — 方法论骨架

**分散 vs 集中的对比：**

| 状态 | 结果 | 原因 |
|------|------|------|
| 无所不备 | 无所不寡 | 兵力分散，处处薄弱 |
| 使人备己 | 众 | 集中兵力，迫使对手分散 |

**核心洞察：**
- 被动防守的人，永远兵力不够
- 主动进攻的人，往往以少胜多
- "专"是力量，"分"是劣势

**现代应用：**
```
被动防守 = 产品线太多 = 资源分散 = 每条线都不强
主动进攻 = 聚焦一点 = 集中突破 = 形成优势
```

## A1 — 书中案例

韩信点兵：
- "韩信点兵，多多益善" 的前提是"善用兵"
- 韩信的策略：集中兵力打穿一点
- 不求每处都强，只求一处必胜

## A2 — 未来触发情境
- 🎯 产品线太多，不知道该聚焦哪里时
- 🎯 资源有限但想做的事太多时
- 🎯 感觉处处都是敌人、防不胜防时
- 🎯 想从"以少敌多"中找到突破口时
- 🎯 团队扩张后反而效率下降时

## E — 执行步骤

### 步骤1：画"备"的地图
你现在在哪些地方都有投入？
- 产品线
- 客户类型
- 渠道
- 地域

### 步骤2：找"最关键的一点"
哪个点的突破能带动全局？
- 不是"最重要"的
- 而是"最能影响其他"的

### 步骤3：集中突破
把资源从其他地方撤出来
- 撤的标准：短期影响可承受
- 投的标准：能形成局部绝对优势

### 步骤4：让对手"无所不备"
你的主动进攻，迫使对手分散兵力
- 你聚焦一点
- 对手必须到处设防
- 对手自然变弱

## B — 边界（什么时候不适用）

❌ **边界1：** 需要多线作战时
- 有些业务天然需要分散
- 多元化集团不能"集中"

❌ **边界2：** 风险分散需要
- 投资组合需要分散
- "集中"意味着高风险

❌ **边界3：** 客户需求差异大
- 如果客户需求差异很大
- 强行集中可能丢失客户
