# Skill: 避实击虚框架

## R — 原文引用
> "夫兵形象水，水之形，避高而趋下，兵之形，避实而击虚。水因地而制流，兵因敌而制胜。"
> ——孙子·虚实第六

## I — 方法论骨架

**"避实击虚"三层含义：**

1. **水的智慧** — 水永远走阻力最小的路
   - 不是"硬冲"，而是"绕过去"
   - 创业同理：与其教育市场，不如顺应市场

2. **虚实判断** — "实"和"虚"是动态的
   - 对手强的地方 ≠ 一直强
   - 对手认为"不重要"的地方 = 可突破的虚

3. **因敌制胜** — 策略没有固定答案
   - 唯一的原则是"根据敌人情况调整"
   - 照搬他人成功经验往往是失败的开端

**核心公式：**
```
竞争优势 = 规避竞争（找对手不关注的地方）
          + 集中力量（在自己优势领域深耕）
          + 快速迭代（敌变我变）
```

## A1 — 书中案例

孙武在虚实第六中描述的战术：
- "我专为一，敌分为十" — 我集中，敌分散
- "以十攻其一" — 用绝对优势兵力打局部
- "敌虽众，能使无斗" — 敌人无法有效抵抗

**商业案例：** 拼多多崛起路径
- 阿里/京东主攻"消费升级"（实）
- 拼多多选择"下沉市场"（虚）
- 阿里京东反应过来时，拼多多已站稳脚跟

## A2 — 未来触发情境
- 🎯 巨头进入你的赛道，直接竞争时
- 🎯 团队资源有限，无法四面出击时
- 🎯 产品推广遇阻，正面战场打不开局面时
- 🎯 职业发展遇到"玻璃天花板"，无法突破时
- 🎯 投资组合需要配置，需要找差异化机会时

## E — 执行步骤

### 步骤1：画"行业虚实地图"
| | 巨头重兵投入（实） | 巨头忽视（虚） |
|---|---|---|
| 我方优势 | 风险区（不碰） | 机会区✓ |
| 我方劣势 | 禁区（绝对不碰） | 观察区 |

### 步骤2：找"一卒可胜"的切入点
- 在"虚"的区域中，找一个足够锋利的点
- 标准：对手利润最高、最怕失去、最难复制的点
- 集中全部资源，打造"非对称优势"

### 步骤3：建立"敌变我变"机制
- 每季度复盘：对手是否注意到我？
- 盯住对手的"新布局"，找新的"虚"
- 保持敏捷，不在旧战场恋战

## B — 边界（什么时候不适用）

❌ **边界1：** 处于"定义标准"阶段
- 如果你能定义行业标准，就不需要避实击虚
- 此时应该"造势"而非"顺势"

❌ **边界2：** 资源充裕到可以"正面碾压"
- 当你有足够资本时，直接收购比迂回更高效
- 但这种情况对99%的创业者不适用

❌ **边界3：** "虚"是陷阱时
- 对手故意示弱，引诱你进入
- 判断方法：看对手在这个领域是否有"沉没成本"
