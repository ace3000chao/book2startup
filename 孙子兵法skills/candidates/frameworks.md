# 《孙子兵法》思维框架候选集

> 提取标准：具有名字的思考方式 / 可迁移的通用流程 / 反复出现的思维结构 / 结构化句式
> 提取原则：不做筛选，宁多勿漏；每条必须有原文根据

- id: f01
  title: 五事七计评估框架
  type: framework
  source_chapter: 始计第一
  source_quote: "故经之以五事，校之以计，而索其情。一曰道，二曰天，三曰地，四曰将，五曰法。曰：主孰有道？将孰有能？天地孰得？法令孰行？兵众孰强？士卒孰练？赏罚孰明？"
  summary: 战前从五个维度（道、天、地、将、法）评估己方实力，再通过七个问题（主孰有道…赏罚孰明）对比敌我优劣，从而预判胜负。是"庙算"的核心操作流程。
  tags: [strategic-planning, competitive-analysis, pre-war-assessment]

- id: f02
  title: 诡道十二法
  type: framework
  source_chapter: 始计第一
  source_quote: "兵者，诡道也。故能而示之不能，用而示之不用，近而示之远，远而示之近。利而诱之，乱而取之，实而备之，强而避之。怒而挠之，卑而骄之。佚而劳之，亲而离之。攻其无备，出其不意。"
  summary: 通过十二种示形方法（示弱、示强、示远、示近等）误导敌人，使其产生错误判断，从而为我方创造进攻机会。是"诡道"的核心操作集。
  tags: [deception, tactics, information-warfare]

- id: f03
  title: 谋攻四层楼
  type: framework
  source_chapter: 谋攻第三
  source_quote: "故上兵伐谋，其次伐交，其次伐兵，其下攻城。"
  summary: 战争策略的四个层次：最高是通过谋略屈敌，其次是通过外交孤立敌人，再次是通过野战击败敌人，最低是攻城。每上升一层消耗越大，故应优先用谋。
  tags: [strategy, conflict-resolution, cost-benefit]

- id: f04
  title: 知胜五法
  type: framework
  source_chapter: 谋攻第三
  source_quote: "故知胜有五：知可以与战不可以与战者，胜；识众寡之用者，胜；上下同欲者，胜；以虞待不虞者，胜；将能而君不御者，胜。"
  summary: 预判胜利的五个条件：知己知彼懂得取舍、知道众寡如何配置、上下目标一致、以有备对无备、将领有能君主不掣肘。满足则胜，不满足则慎战。
  tags: [pre-war-assessment, victory-conditions, leadership]

- id: f05
  title: 知彼知己框架
  type: framework
  source_chapter: 谋攻第三
  source_quote: "故曰：知彼知己，百战不殆。不知彼而知己，一胜一负。不知彼不知己，每战必败。"
  summary: 根据对彼/己的了解程度将胜率分为三档：既知彼又知己则必胜；只知己不知彼则胜负各半；既不知彼又不知己则每战必败。是贯穿全书的方法论核心。
  tags: [intelligence, self-awareness, decision-making]

- id: f06
  title: 先为不可胜框架
  type: framework
  source_chapter: 军形第四
  source_quote: "昔之善战者，先为不可胜，以待敌之可胜。不可胜在己，可胜在敌。故善战者，能为不可胜，不能使敌之必可胜。故曰：胜可知而不可为。"
  summary: 最佳策略是首先确保己方不可被战胜（防守），然后等待敌人出现可乘之机（进攻时机）。己方不可被战胜取决于自己，敌人是否可被战胜取决于敌人。
  tags: [defensive-strategy, patience, self-reliance]

- id: f07
  title: 胜兵败兵框架
  type: framework
  source_chapter: 军形第四
  source_quote: "是故胜兵先胜而后求战，败兵先战而后求胜。"
  summary: 胜者先创造必胜条件再开战，败者先开战再寄希望于侥幸。以战求胜者被动，以势取胜者主动。先胜后战是稳健之道。
  tags: [strategic-planning, risk-management, preparation]

- id: f08
  title: 度量数称胜框架
  type: framework
  source_chapter: 军形第四
  source_quote: "兵法：一曰度，二曰量，三曰数，四曰称，五曰胜。地生度，度生量，量生数，数生称，称生胜。"
  summary: 从"度"（测量形势）到"胜"（夺取胜利）的五步推导链：形势决定攻守角色，攻守角色决定优劣分析，优劣分析决定方略制定，方略制定决定胜负预测。
  tags: [logical-reasoning, strategic-planning, decision-chain]

- id: f09
  title: 奇正相生框架
  type: framework
  source_chapter: 兵势第五
  source_quote: "凡战者，以正合，以奇胜。战势不过奇正，奇正之变，不可胜穷也。奇正相生，如循环之无端，孰能穷之哉？"
  summary: 正面部队交战吸引注意，侧翼奇兵突击取胜。奇正不是固定不变，而是相互转化、循环无穷。善出奇者战术变化如天地江海，永不枯竭。
  tags: [tactics, flexibility, asymmetric-warfare]

- id: f10
  title: 势险节短框架
  type: framework
  source_chapter: 兵势第五
  source_quote: "激水之疾，至于漂石者，势也；鸷鸟之疾，至于毁折者，节也。故善战者，其势险，其节短。势如弩，节如发机。"
  summary: 强大攻势需要两个条件——势险（力量蓄积如满弓）和节短（攻击距离近、节奏快）。像水漂石、鹰毁折一样，在最短距离内释放最大力量。
  tags: [tactics, power-release, timing]

- id: f11
  title: 致人而不致于人框架
  type: framework
  source_chapter: 虚实第六
  source_quote: "故善战者，致人而不致于人。能使敌人自至者，利之也；能使敌人不得至者，害之也。"
  summary: 核心是掌握战争主动权：通过利益诱惑调动敌人主动来犯，或通过攻击要害阻止敌人接近。被调动者被动挨打，调动他人者掌控全局。
  tags: [initiative, command-control, strategic-advantage]

- id: f12
  title: 避实击虚框架
  type: framework
  source_chapter: 虚实第六
  source_quote: "夫兵形象水，水之形，避高而趋下，兵之形，避实而击虚。水因地而制流，兵因敌而制胜。"
  summary: 战争的规律如水：水流自动避高就低，军队自动避实击虚。没有固定的作战方式，而是根据敌情随时调整策略。因敌制胜者是谓"神"。
  tags: [tactics, adaptability, asymmetric-advantage]

- id: f13
  title: 形人而我无形框架
  type: framework
  source_chapter: 虚实第六
  source_quote: "故形人而我无形，则我专而敌分。我专为一，敌分为十，是以十攻其一也，则我众敌寡。"
  summary: 让敌人暴露行迹而我方隐藏行迹，使敌人分散兵力而我方集中兵力。通过虚实变换，实现局部以多击少。敌人无处不防则无处不寡。
  tags: [deception, force-concentration, information-control]

- id: f14
  title: 因敌变化框架
  type: framework
  source_chapter: 虚实第六
  source_quote: "故兵无常势，水无常形，能因敌变化而取胜者，谓之神。故五行无常胜，四时无常位，日有短长，月有死生。"
  summary: 没有任何策略是常胜的，就像五行没有常胜、四季没有常在。真正的高手能根据敌人变化而灵活调整，从不重复相同的战法。
  tags: [adaptability, flexibility, situational-awareness]

- id: f15
  title: 以迂为直框架
  type: framework
  source_chapter: 军争第七
  source_quote: "军争之难者，以迂为直，以患为利。故迂其途而诱之以利，后人发，先人至，此知迂直之计者也。"
  summary: 把弯路变成直路，把不利变成有利。故意走远路麻痹敌人，在敌人放松警惕时实现"后发先至"。通过迷惑敌人获得战略先机。
  tags: [strategy, deception, indirect-approach]

- id: f16
  title: 兵之情主速框架
  type: framework
  source_chapter: 军争第七 / 九地第十一
  source_quote: "兵之情主速，乘人之不及，由不虞之道，攻其所不戒也。"
  summary: 作战的核心法则是速度：在敌人来不及准备时到达，从敌人意想不到的道路进攻，在敌人没有戒备的地方发动攻击。速战是所有战略的核心。
  tags: [speed, surprise, operational-efficiency]

- id: f17
  title: 治气治心治力治变框架
  type: framework
  source_chapter: 军争第七
  source_quote: "善用兵者，避其锐气，击其惰归，此治气者也；以治待乱，以静待哗，此治心者也；以近待远，以逸待劳，以饱待饥，此治力者也；无邀正正之旗，勿击堂堂之阵，此治变者也。"
  summary: 从四个维度强化己方、削弱敌方：治气（避锐击惰）、治心（以静待哗）、治力（以逸待劳）、治变（不击强阵）。核心是以最小代价获得最大优势。
  tags: [morale-management, self-strengthening, tactics]

- id: f18
  title: 杂于利害框架
  type: framework
  source_chapter: 九变第八
  source_quote: "是故智者之虑，必杂于利害。杂于利而务可信也，杂于害而患可解也。"
  summary: 聪明人考虑问题一定同时兼顾利与害：在有利时想到潜在危害以确保成功，在有害时看到转化机会以解除祸患。孤立看待利或害都是危险的。
  tags: [decision-making, risk-management,辩证思维]

- id: f19
  title: 杂于利害驱动他人框架
  type: framework
  source_chapter: 九变第八
  source_quote: "是故屈诸侯者以害，役诸侯者以业，趋诸侯者以利。"
  summary: 用害处迫使诸侯屈服，用事务驱使诸侯疲惫，用利益诱导诸侯行动。三种手段对应三种控制目标，是博弈中的基本驱动框架。
  tags: [influence,博弈策略, leadership]

- id: f20
  title: 恃吾有以待之框架
  type: framework
  source_chapter: 九变第八
  source_quote: "无恃其不来，恃吾有以待之；无恃其不攻，恃吾有所不可攻也。"
  summary: 不依赖敌人不来侵犯，而是依靠自己的充分准备；不依赖敌人不进攻，而是依靠自己不可被攻破的实力。核心是反求诸己而非寄望于敌。
  tags: [preparation, self-reliance, defensive-strategy]

- id: f21
  title: 五危框架
  type: framework
  source_chapter: 九变第八
  source_quote: "故将有五危：必死可杀，必生可掳，忿速可侮，廉洁可辱，爱民可烦。"
  summary: 将领有五种危险特质：一味死战易被杀，贪生怕死易被俘，急躁易怒易被激，过于廉洁易被辱，过度爱民易被扰。这五点是败将的共性特征。
  tags: [leadership, self-awareness, failure-prevention]

- id: f22
  title: 处军四法
  type: framework
  source_chapter: 行军第九
  source_quote: "凡处军相敌，绝山依谷，视生处高，战隆无登，此处山之军也。绝水必远水…此处水上之军也。绝斥泽…此处斥泽之军也。平陆处易，而右背高，前死后生，此处平陆之军也。"
  summary: 在山地、河流、沼泽、平原四种地形中行军布阵的原则：必处高向阳、背靠依托。根据地形特性选择驻扎和作战方式，是地形运用的基本框架。
  tags: [terrain, military-operations, logistics]

- id: f23
  title: 相敌三十二法
  type: framework
  source_chapter: 行军第九
  source_quote: "近而静者，恃其险也；远而挑战者，欲人之进也…杖而立者，饥也；汲而先饮者，渴也…辞卑而益备者，进也；辞强而进趋者，退也…"
  summary: 通过观察敌人的外在表现（动静、尘土、使者言辞、士兵状态等32种征候）推断敌军真实情况。是战场侦察和商业情报收集的基础方法论。
  tags: [intelligence, reconnaissance, observation]

- id: f24
  title: 令之以文齐之以武框架
  type: framework
  source_chapter: 行军第九
  source_quote: "故令之以文，齐之以武，是谓必取。令素行，以教其民，则民服；令不素行，以教其民，则民不服。"
  summary: 用仁德感化士兵（文），用军法统一行动（武），恩威并施才能让士兵信服。政令平时一贯执行，教化百姓才能服从；令行禁止是领导力的基础。
  tags: [leadership, military-management, organizational-culture]

- id: f25
  title: 六地法则
  type: framework
  source_chapter: 地形第十
  source_quote: "地形有通者，有挂者，有支者，有隘者，有险者，有远者…通形者先居高阳…挂形者敌无备出而胜之…支形者敌虽利我我无出也…"
  summary: 根据六种地形（通、挂、支、隘、险、远）制定不同的战术决策：选择驻扎方向、决定攻守策略、选择进退时机。地形是战争的重要辅助条件。
  tags: [terrain, tactical-decisions, geography]

- id: f26
  title: 败之道六种
  type: framework
  source_chapter: 地形第十
  source_quote: "故兵有走者，有弛者，有陷者，有崩者，有乱者，有北者。凡此六者，非天地之灾，将之过也。"
  summary: 失败的六种模式：走（以寡敌众必然逃）、弛（卒强吏弱必然散）、陷（吏强卒弱必然败）、崩（裨将不服必然溃）、乱（将弱政不明必然乱）、北（无选锋必然败）。六种皆是将领之过。
  tags: [failure-analysis, military-management, leadership]

- id: f27
  title: 知彼知己知天知地框架
  type: framework
  source_chapter: 地形第十
  source_quote: "知吾卒之可以击，而不知敌之不可击，胜之半也；知敌之可击，而不知吾卒之不可以击，胜之半也…故曰：知彼知己，胜乃不殆；知天知地，胜乃可全。"
  summary: 在"知彼知己"基础上增加"知天知地"才能全胜。知己不知彼（50%胜率）、知彼不知己（50%胜率）、知己知彼不知天地（有风险）。四知一体才是完整的决策基础。
  tags: [intelligence, comprehensive-awareness, decision-making]

- id: f28
  title: 九地法则
  type: framework
  source_chapter: 九地第十一
  source_quote: "用兵之法，有散地，有轻地，有争地，有交地，有衢地，有重地，有圮地，有围地，有死地。是故散地则无战，轻地则无止，争地则无攻，交地则无绝，衢地则合交，重地则掠，圮地则行，围地则谋，死地则战。"
  summary: 根据进入敌境的深度和所处形势，划分九种地境（散、轻、争、交、衢、重、圮、围、死），每种都有对应的行动原则：散地统一意志、死地拼死力战等。
  tags: [strategic-positions, military-operations, situation-management]

- id: f29
  title: 合于利而动框架
  type: framework
  source_chapter: 九地第十一
  source_quote: "合于利而动，不合于利而止。"
  summary: 所有军事行动的唯一判断标准是"是否有利于己"。有利则动，不利则止。不因愤怒、贪婪或面子而战。理性的决策过滤器，贯穿所有战略层次。
  tags: [decision-making, rational-action, strategic-discipline]

- id: f30
  title: 率然法则
  type: framework
  source_chapter: 九地第十一
  source_quote: "故善用兵者，譬如率然。率然者，常山之蛇也，击其首则尾至，击其尾则首至，击其中则首尾俱至。"
  summary: 理想的军队如"率然蛇"——首尾呼应、相互救援。能让军队做到这一点的关键是"齐勇若一"（政之道）和"刚柔皆得"（地之理）。协同一致是战斗力的核心。
  tags: [coordination, military-management, teamwork]

- id: f31
  title: 愚士卒耳目框架
  type: framework
  source_chapter: 九地第十一
  source_quote: "能愚士卒之耳目，使之无知。易其事，革其谋，使人无识；易其居，迂其途，使人不得虑。"
  summary: 战时让士兵"无知"（不知道真实情况）的管理方法：不断改变任务和计划使敌人无法预测，故意绕远路使敌人无法预判。是控制信息不对称的极端手段。
  tags: [information-control, military-management, leadership]

- id: f32
  title: 将将静幽正治框架
  type: framework
  source_chapter: 九地第十一
  source_quote: "将军之事，静以幽，正以治。"
  summary: 将领的行事风格：静（沉稳不妄动）、幽（幽深难测）、正（公正不偏私）、治（整治有方）。静是外在表现，幽是威慑来源，正是赢得信任，治是管理能力。
  tags: [leadership, military-management, personal-effectiveness]

- id: f33
  title: 投之亡地框架
  type: framework
  source_chapter: 九地第十一
  source_quote: "投之亡地然后存，陷之死地然后生。夫众陷于害，然后能为胜败。"
  summary: 把军队置于无路可退的绝境，反而能激发其拼死求生的最大战斗力。"亡地"倒逼出"存"的意志，"死地"激发出"生"的可能。绝境是激发潜力的手段。
  tags: [motivation, crisis-management,极限思维]

- id: f34
  title: 顺详敌意框架
  type: framework
  source_chapter: 九地第十一
  source_quote: "故为兵之事，在于顺详敌之意，并敌一向，千里杀将，此谓巧能成事者也。是故始如处女，敌人开户，后如脱兔，敌不及拒。"
  summary: 先假装顺从敌人的意图（处女般的安静），等敌人放松警惕后（开户），突然集中兵力全力一击（脱兔）。表面顺从实则暗度陈仓的典型策略。
  tags: [deception, patience, sudden-strike]

- id: f35
  title: 五火之变框架
  type: framework
  source_chapter: 火攻第十二
  source_quote: "凡火攻有五：一曰火人，二曰火积，三曰火辎，四曰火库，五曰火队。凡火攻，必因五火之变而应之。"
  summary: 火攻的五种目标（烧人、烧积、烧辎重、烧仓库、烧器械），每种需要相应的部队配合。火攻不是单独行动，必须根据火势变化安排接应部队。
  tags: [military-operations, combined-arms, tactical-planning]

- id: f36
  title: 慎战原则
  type: framework
  source_chapter: 火攻第十二
  source_quote: "非利不动，非得不用，非危不战。主不可以怒而兴师，将不可以愠而致战。合于利而动，不合于利而止。"
  summary: 战争的三大启动条件：非利不动（无利益不行动）、非得不用（无把握不开战）、非危不战（非危急不拼命）。更不能因君主愤怒或将领怨恨而开战。
  tags: [strategic-discipline, war-ethics, rational-decision]

  title: 五间俱起框架
  type: framework
  source_chapter: 用间第十三
  source_quote: "故用间有五：有因间，有内间，有反间，有死间，有生间。五间俱起，莫知其道，是谓神纪，人君之重宝也。"
  summary: 五种间谍同时使用（乡间、内间、反间、死间、生间），使敌人无法判断我方情报来源和策略，这是最神奇的情战法则。反间是五间核心，因为能获取最核心情报。
  tags: [intelligence, espionage, information-warfare]

- id: f38
  title: 先知框架
  type: framework
  source_chapter: 用间第十三
  source_quote: "故明君贤将，所以动而胜人，成功出于众者，先知也。先知者，不可取于鬼神，不可象于事，不可验于度，必取于人，知敌之情者也。"
  summary: 胜利来自"先知"——提前掌握敌情。先知不能靠鬼神占卜、不能靠类比推测、不能靠仪器测量，只能依靠人的情报工作。情报是战争胜利的基础。
  tags: [intelligence, evidence-based, information-acquisition]

- id: f39
  title: 庙算胜负框架
  type: framework
  source_chapter: 始计第一
  source_quote: "夫未战而庙算胜者，得算多也；未战而庙算不胜者，得算少也。多算胜，少算不胜，而况于无算乎？"
  summary: 战前在庙堂上谋划时胜负已定：计算周全（多算）则胜，计算不足（少算）则败，不计算则必败。庙算是战争决策的第一道关卡。
  tags: [strategic-planning, pre-war-assessment, calculation]
