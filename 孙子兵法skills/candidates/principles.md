# 孙子兵法——原则候选清单

## 说明
本文件从《孙子兵法》原文及注释中提取所有具有"规则/断言/原则"性质的条目，
不做筛选，全部收录。每条均标注原文出处，确保可回溯。

---

- id: p01
  title: 战前必察五事
  type: principle
  source_chapter: 始计第一
  source_quote: "孙子曰：兵者，国之大事，死生之地，存亡之道，不可不察也。故经之以五事，校之以计，而索其情。一曰道，二曰天，三曰地，四曰将，五曰法。"
  summary: 战争关乎生死存亡，开战前必须从"道天地将法"五个维度系统分析敌我情况。
  tags: [战略规划,庙算,五事]

- id: p02
  title: 将领必备五德
  type: principle
  source_chapter: 始计第一
  source_quote: "将者，智、信、仁、勇、严也。……知之者胜，不知者不胜。"
  summary: 合格将领必须同时具备智慧、信用、仁德、勇气、威严五项特质，缺一不可。
  tags: [领导力,将领素质]

- id: p03
  title: 庙算多者胜
  type: principle
  source_chapter: 始计第一
  source_quote: "夫未战而庙算胜者，得算多也；未战而庙算不胜者，得算少也。多算胜，少算不胜，而况于无算乎？"
  summary: 战前在朝堂上筹划越充分，获胜概率越高；不筹划就贸然出兵必败。
  tags: [庙算,战略规划,先胜]

- id: p04
  title: 兵者诡道——示形惑敌
  type: principle
  source_chapter: 始计第一
  source_quote: "兵者，诡道也。故能而示之不能，用而示之不用，近而示之远，远而示之近。"
  summary: 战争以计谋为核心，必须通过示形（示弱、示强、示远、示近）迷惑敌人。
  tags: [诡道,欺骗,战术]

- id: p05
  title: 因利制权，随机应变
  type: principle
  source_chapter: 始计第一
  source_quote: "势者，因利而制权也。"
  summary: "势"是根据有利条件灵活变通，根据敌我优劣随时调整策略。
  tags: [权变,形势]

- id: p06
  title: 八诡诱敌法
  type: principle
  source_chapter: 始计第一
  source_quote: "利而诱之，乱而取之，实而备之，强而避之，怒而挠之，卑而骄之，佚而劳之，亲而离之。攻其无备，出其不意。"
  summary: 通过八种诡道手段（利诱、混乱、备实、避强、激挠、骄敌、劳敌、离间）调动敌人，在其无备时不意中攻击。
  tags: [诡道,战术,欺骗]

- id: p07
  title: 兵贵速不贵久
  type: principle
  source_chapter: 作战第二
  source_quote: "其用战也，胜久则钝兵挫锐，攻城则力屈，久暴师则国用不足。……故兵闻拙速，未睹巧之久也。"
  summary: 持久战消耗国力，拖垮士气。宁可计谋简单也要速战，绝不能智谋深厚却久拖不决。
  tags: [速战,持久战,消耗]

- id: p08
  title: 不尽知用兵之害，则不能尽知用兵之利
  type: principle
  source_chapter: 作战第二
  source_quote: "夫兵久而国利者，未之有也。故不尽知用兵之害者，则不能尽知用兵之利也。"
  summary: 不深刻理解战争的所有危害，就无法真正理解战争的所有好处。必须全面评估风险才能做出正确决策。
  tags: [风险评估,慎战,利弊]

- id: p09
  title: 因粮于敌
  type: principle
  source_chapter: 作战第二
  source_quote: "取用于国，因粮于敌，故军食可足也。……食敌一钟，当吾二十钟。秆一石，当吾二十石。"
  summary: 从敌人那里获取物资，相当于节省国内二十倍运力。以战养战是减少己方消耗的核心方法。
  tags: [后勤,资源,因粮于敌]

- id: p10
  title: 杀敌者怒也，取敌者货也
  type: principle
  source_chapter: 作战第二
  source_quote: "故杀敌者，怒也。取敌之利者，货也。……是谓胜敌而益强。"
  summary: 激励士兵奋勇杀敌靠激发怒气，激励缴获敌人财物靠物质奖励。善待俘虏，使其为我所用，战胜敌人后让自己更强。
  tags: [激励,士气,俘虏政策]

- id: p11
  title: 兵贵胜不贵久
  type: principle
  source_chapter: 作战第二
  source_quote: "故兵贵胜不贵久。故知兵之将，民之司命，国家安危之主也。"
  summary: 战争追求的是胜利而非拖延。懂得战争的将领主宰百姓生死和国家安危。
  tags: [速战,将领责任]

- id: p12
  title: 全胜为上，破胜次之
  type: principle
  source_chapter: 谋攻第三
  source_quote: "凡用兵之法，全国为上，破国次之；全军为上，破军次之；……是不战而屈人之兵，善之善者也。"
  summary: 战争最高境界是保全敌人（国家/军队）使其屈服，其次才是攻破。百战百胜并非最高明的，不战而屈人才是。
  tags: [全胜,战略目标,不战]

- id: p13
  title: 上兵伐谋，其次伐交，其次伐兵，其下攻城
  type: principle
  source_chapter: 谋攻第三
  source_quote: "故上兵伐谋，其次伐交，其次伐兵，其下攻城。攻城之法，为不得已。"
  summary: 战胜敌人的优先顺序是：谋略、外交、正战，最次是攻城。攻城是不得已的手段，代价极大。
  tags: [战略层次,伐谋,攻城]

- id: p14
  title: 十则围之，五则攻之
  type: principle
  source_chapter: 谋攻第三
  source_quote: "故用兵之法，十则围之，五则攻之，倍则分之。敌则能战之，少则能逃之，不若则能避之。故小敌之坚，大敌之擒也。"
  summary: 根据敌我兵力对比选择战术：十倍围之，五倍攻之，两倍分敌；兵力相当则战，不足则逃，不若则避。以弱抗强必被擒。
  tags: [兵力对比,战术选择]

- id: p15
  title: 君命有所不受
  type: principle
  source_chapter: 谋攻第三
  source_quote: "三军既惑且疑，则诸侯之难至矣，是谓乱军引胜。……将能而君不御者，胜。"
  summary: 国君不懂军事而遥控指挥会祸害军队。将军有智慧有能力，国君不横加干涉，才能获胜。
  tags: [授权,君臣关系,将能]

- id: p16
  title: 知胜有五
  type: principle
  source_chapter: 谋攻第三
  source_quote: "知胜有五：知可以与战不可以与战者，胜；识众寡之用者，胜；上下同欲者，胜；以虞待不虞者，胜；将能而君不御者，胜。"
  summary: 可预见的五种胜利条件：知战与不战、懂众寡战术、上下同心、以有备对无备、将能君不御。
  tags: [知胜,胜利条件]

- id: p17
  title: 知己知彼，百战不殆
  type: principle
  source_chapter: 谋攻第三
  source_quote: "故曰：知彼知己，百战不殆。不知彼而知己，一胜一负。不知彼不知己，每战必败。"
  summary: 了解敌我双方情况才能常胜不危；只知己不知彼胜负各半；两者都不知则每战必败。
  tags: [知己知彼,核心原则]

- id: p18
  title: 先为不可胜，以待敌之可胜
  type: principle
  source_chapter: 军形第四
  source_quote: "昔之善战者，先为不可胜，以待敌之可胜。不可胜在己，可胜在敌。"
  summary: 善战者首先确保自己不可被战胜，然后等待敌人出现可乘之机。防守靠自己，胜利靠敌人犯错。
  tags: [先胜,防守,不可胜]

- id: p19
  title: 善战者胜于易胜
  type: principle
  source_chapter: 军形第四
  source_quote: "古之所谓善战者，胜于易胜者也。故善战者之胜也，无智名，无勇功。"
  summary: 真正善战者选择容易打败的敌人取胜，看似无功无名，实则大巧若拙。每战必胜源于在敌人已有败相时出击。
  tags: [易胜,善战]

- id: p20
  title: 善守者藏于九地之下，善攻者动于九天之上
  type: principle
  source_chapter: 军形第四
  source_quote: "善守者藏于九地之下，善攻者动于九天之上，故能自保而全胜也。"
  summary: 善守者借地形隐蔽如藏于九地之下；善攻者借天时迅猛如动于九天之上，两者结合才能自保且全胜。
  tags: [攻守,形势]

- id: p21
  title: 修道而保法
  type: principle
  source_chapter: 军形第四
  source_quote: "善用兵者，修道而保法，故能为胜败之政。"
  summary: 善于用兵者，既丰富谋略（修道）又严守军法（保法），两者结合才能掌握胜败之权。
  tags: [修道保法,领导力]

- id: p22
  title: 胜兵先胜而后求战
  type: principle
  source_chapter: 军形第四
  source_quote: "是故胜兵先胜而后求战，败兵先战而后求胜。"
  summary: 胜利的军队先有必胜把握再战；失败的军队先战再寄希望于侥幸取胜。
  tags: [先胜后战,战略]

- id: p23
  title: 以正合，以奇胜
  type: principle
  source_chapter: 兵势第五
  source_quote: "凡战者，以正合，以奇胜。故善出奇者，无穷如天地，不竭如江海。……奇正相生，如循环之无端，孰能穷之哉？"
  summary: 正面迎敌用正兵，出奇制胜用奇兵。奇正相互转化，变化无穷无尽。
  tags: [奇正,战术变化]

- id: p24
  title: 其势险，其节短
  type: principle
  source_chapter: 兵势第五
  source_quote: "故善战者，其势险，其节短。势如弩，节如发机。"
  summary: 善战者进攻势头猛烈（势险），攻击距离极短（节短）。蓄势如满弓，行动如扣发扳机。
  tags: [势,战术节奏]

- id: p25
  title: 择人而任势
  type: principle
  source_chapter: 兵势第五
  source_quote: "故善战者，求之于势，不责于人。故能择人而任势。"
  summary: 善战者依靠形势权变取胜，不苛责下属。善于根据不同形势选用不同人才。
  tags: [任势,领导力]

- id: p26
  title: 乱生于治，怯生于勇，弱生于强
  type: principle
  source_chapter: 兵势第五
  source_quote: "乱生于治，怯生于勇，弱生于强。治乱，数也；勇怯，势也；强弱，形也。"
  summary: 混乱的假象源于严整；胆怯的假象源于勇敢；弱小的假象源于强大。治乱取决于编制，勇怯取决于态势，强弱取决于外在形迹。
  tags: [示形,诡道,辩证]

- id: p27
  title: 致人而不致于人
  type: principle
  source_chapter: 虚实第六
  source_quote: "故善战者，致人而不致于人。能使敌人自至者，利之也；能使敌人不得至者，害之也。"
  summary: 善战者调动敌人而不被敌人调动。用利益引诱敌人，用要害威胁阻止敌人。
  tags: [主动权,致人]

- id: p28
  title: 佚能劳之，饱能饥之，安能动之
  type: principle
  source_chapter: 虚实第六
  source_quote: "故敌佚能劳之，饱能饥之，安能动之。出其所不趋，趋其所不意。"
  summary: 敌人安逸就使之疲劳；敌人饱食就使之饥饿；敌人安静就调动它。进攻敌人无法救援之地，在敌人意想不到之时出兵。
  tags: [虚实,调动敌人]

- id: p29
  title: 善攻者敌不知其所守，善守者敌不知其所攻
  type: principle
  source_chapter: 虚实第六
  source_quote: "故善攻者，敌不知其所守；善守者，敌不知其所攻。"
  summary: 善于进攻者让敌人不知该在何处布防；善于防守者让敌人不知该从何处进攻。
  tags: [攻守,虚实]

- id: p30
  title: 我专而敌分
  type: principle
  source_chapter: 虚实第六
  source_quote: "故形人而我无形，则我专而敌分，我专为一，敌分为十，是以十攻其一也，则我众敌寡。"
  summary: 设法暴露敌人行迹同时隐藏己方行迹，我方集中而敌人分散，从而实现以十攻一，以众击寡。
  tags: [虚实,集中兵力]

- id: p31
  title: 无所不备则无所不寡
  type: principle
  source_chapter: 虚实第六
  source_quote: "故备前则后寡，备后则前寡，备左则右寡，备右则左寡，无所不备，则无所不寡。寡者，备人者也；众者，使人备己者也。"
  summary: 敌人到处设防就会到处薄弱。兵力少是因为被动防备；兵力多是因为让对方处处防备自己。
  tags: [虚实,集中与分散]

- id: p32
  title: 兵形象水——因敌制胜
  type: principle
  source_chapter: 虚实第六
  source_quote: "夫兵形象水，水之形，避高而趋下，兵之形，避实而击虚。水因地而制流，兵因敌而制胜。故兵无常势，水无常形，能因敌变化而取胜者，谓之神。"
  summary: 战争态势如水流——避高趋下，避实击虚。水因地形改变流向，兵因敌情调整战略。能根据敌变而取胜者为神。
  tags: [因敌制胜,权变,核心比喻]

- id: p33
  title: 以迂为直
  type: principle
  source_chapter: 军争第七
  source_quote: "军争之难者，以迂为直，以患为利。故迂其途而诱之以利，后人发，先人至，此知迂直之计者也。"
  summary: 军争最难之处在于将曲折变为直路，将不利变为有利。故意走远路诱敌，虽后出发却先到达。
  tags: [迂直,军争,战术]

- id: p34
  title: 兵以诈立，以利动，以分合为变
  type: principle
  source_chapter: 军争第七
  source_quote: "故兵以诈立，以利动，以分合为变者也。"
  summary: 战争通过计谋确立优势，通过利益调动部队，通过分兵合兵实现战术变化。
  tags: [诡道,战术变化]

- id: p35
  title: 军队无辎重则亡
  type: principle
  source_chapter: 军争第七
  source_quote: "是故军无辎重则亡，无粮食则亡，无委积则亡。"
  summary: 军队没有器械车马会灭亡，没有粮食会灭亡，没有物资储备会灭亡。
  tags: [后勤,辎重]

- id: p36
  title: 三军可夺气，将军可夺心
  type: principle
  source_chapter: 军争第七
  source_quote: "故三军可夺气，将军可夺心。是故朝气锐，昼气惰，暮气归。善用兵者，避其锐气，击其惰归，此治气者也。"
  summary: 士气可被削弱，将军心神可被扰乱。士气晨锐昼衰暮竭，善用兵者避敌锐气击其惰归。
  tags: [士气,治气,心理战]

- id: p37
  title: 以治待乱，以静待哗
  type: principle
  source_chapter: 军争第七
  source_quote: "以治待乱，以静待哗，此治心者也；以近待远，以逸待劳，以饱待饥，此治力者也。"
  summary: 用己方安定对付敌方混乱，用己方冷静对付敌方喧哗，以此掌握军心；用己方近便对付敌方远途，用己方安逸对付敌方疲劳，用己方饱食对付敌方饥饿，以此提升战斗力。
  tags: [以逸待劳,治力]

- id: p38
  title: 勿击堂堂之阵
  type: principle
  source_chapter: 军争第七
  source_quote: "无邀正正之旗，勿击堂堂之阵，此治变者也。"
  summary: 不要阻击旗帜整齐的军队，不要攻击阵容庞大的敌人，这是根据敌情变化而采取的策略。
  tags: [战术判断,治变]

- id: p39
  title: 围师必阙，穷寇勿迫
  type: principle
  source_chapter: 军争第七
  source_quote: "故用兵之法，高陵勿向，背丘勿逆，佯北勿从，锐卒勿攻，饵兵勿食，归师勿遏，围师必阙，穷寇勿迫，此用兵之法也。"
  summary: 八种情况下的正确处置原则：敌人占据高地不要仰攻；假装撤退不要追击；锐气正盛不要进攻；归国敌军不要拦截；包围敌人要留缺口；穷寇不要逼太紧。
  tags: [战术禁忌,围师必阙]

- id: p40
  title: 杂于利害
  type: principle
  source_chapter: 九变第八
  source_quote: "是故智者之虑，必杂于利害，杂于利而务可信也，杂于害而患可解也。"
  summary: 聪明的将军思考问题必须同时兼顾利与害。在有利情况下考虑不利方面，祸患可以消除；在不利情况下看到有利方面，成功才有保障。
  tags: [利弊思维,决策]

- id: p41
  title: 无恃其不来，恃吾有以待之
  type: principle
  source_chapter: 九变第八
  source_quote: "故用兵之法，无恃其不来，恃吾有以待之；无恃其不攻，恃吾有所不可攻也。"
  summary: 不要指望敌人不来侵犯，而是要靠自己充分准备；不要指望敌人不进攻，而是要靠自己防守严密使敌人无法攻克。
  tags: [备战,主动防御]

- id: p42
  title: 将有五危
  type: principle
