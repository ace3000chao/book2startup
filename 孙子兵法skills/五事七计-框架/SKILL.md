# Skill: 五事七计评估框架

## R — 原文引用
> "故经之以五事，校之以计，而索其情。一曰道，二曰天，三曰地，四曰将，五曰法。
> 曰：主孰有道？将孰有能？天地孰得？法令孰行？兵众孰强？士卒孰练？赏罚孰明？"
> ——孙子·始计第一

## I — 方法论骨架

**五事：** 评估的基本维度
| 五事 | 内涵 | 商战对应 |
|------|------|---------|
| 道 | 人心向背 | 企业文化、团队愿景、使命愿景 |
| 天 | 天时时机 | 行业周期、政策环境、市场窗口 |
| 地 | 地理形势 | 市场位置、渠道网络、竞争格局 |
| 将 | 领导者 | 创始人/CEO的能力与特质 |
| 法 | 组织制度 | 管理制度、激励机制、运营流程 |

**七计：** 敌我对比的七个问题
1. **主孰有道** — 谁的愿景使命更得人心？
2. **将孰有能** — 谁的核心领导者更有能力？
3. **天地孰得** — 谁把握的天时地利更好？
4. **法令孰行** — 谁的制度执行更有力？
5. **兵众孰强** — 谁的整体实力更强？
6. **士卒孰练** — 谁的团队训练有素？
7. **赏罚孰明** — 谁的激励约束更清晰？

**核心洞察：**
- 五事是内部分析，七计是横向对比
- "道"排在第一，说明"人心"是根本
- 量化对比才能得出客观结论

## A1 — 书中案例

孙武见吴王阖闾，以五事七计分析吴国vs楚国：
- 吴国：阖闾"有道"，孙武"能将"，本土作战"有地"
- 楚国：楚王无道（内部离心），将相不和（令不行）
- 结论：吴必胜楚（后吴西破强楚）

## A2 — 未来触发情境
- 🎯 创业项目立项前的自我评估
- 🎯 商业计划书中的竞争分析章节
- 🎯 融资路演时的战略表述
- 🎯 团队周会/月度会的战略复盘
- 🎯 跨部门项目立项的资源争取
- 🎯 个人职业发展的核心竞争力评估

## E — 执行步骤

### 步骤1：填五事雷达图
对己方在五个维度打分（1-10分）：
```
        天（外部时机）
          ↑
         /\
        /  \
       /    \
左将←─┤      ├→右法（制度）
       \    /
        \  /
         \/
        地（资源位置）
          ↓
        底：道（文化愿景）
```

### 步骤2：做七计对比表
| 七计问题 | 己方 | 对手 | 差距 |
|---------|------|------|------|
| 主孰有道 | 7 | 5 | +2 |
| 将孰有能 | 6 | 7 | -1 |
| ... | | | |

### 步骤3：制定"补短板"行动计划
- 差距≤2：保持现状，观察
- 差距3-5：制定追赶计划
- 差距>5：考虑战略转向或差异化

## B — 边界（什么时候不适用）

❌ **边界1：** 处于"探索期"的新项目
- 此时五事七计需要的很多信息不存在
- 过早量化分析会扼杀创新探索

❌ **边界2：** 高度依赖融资的阶段
- 资本可以短期弥补"五事"的不足
- 此时"将孰有能"（融资能力）反而最重要

❌ **边界3：** 行业剧变期
- 旧有的"七计"维度可能不再适用
- 颠覆性创新者的"五事"评分往往很低但仍能赢
